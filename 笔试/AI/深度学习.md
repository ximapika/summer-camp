## 一、神经网络基础

### 1.1 感知机（Perceptron）

单层感知机：y = sign(wᵀx + b)

只能处理线性可分问题（无法解决 XOR 问题）。

感知机学习算法：对误分类样本 (x_i, y_i)，更新 w ← w + ηy_ix_i，b ← b + ηy_i。

### 1.2 多层感知机（MLP / Feedforward Neural Network）

通过引入隐藏层和非线性激活函数解决非线性问题。

$$h = \sigma(W_1 x + b_1)$$
$$y = W_2 h + b_2$$

**万能近似定理（Universal Approximation Theorem）**：具有至少一个隐藏层的前馈神经网络，在隐藏层神经元足够多的情况下，可以以任意精度逼近任意连续函数。但定理不保证学习效率。

### 1.3 激活函数（Activation Function）

| 激活函数 | 公式 | 值域 | 特点 |
|---|---|---|---|
| Sigmoid | σ(x) = 1/(1+e⁻ˣ) | (0, 1) | 梯度消失、输出非零中心、计算 exp 开销 |
| Tanh | tanh(x) = (eˣ-e⁻ˣ)/(eˣ+e⁻ˣ) | (-1, 1) | 零中心化，但仍有梯度消失 |
| ReLU | max(0, x) | [0, +inf) | 计算简单，缓解梯度消失，但有 Dead ReLU 问题（神经元永远不激活） |
| Leaky ReLU | max(αx, x), α=0.01 | (-inf, +inf) | 解决 Dead ReLU |
| GELU | x * Φ(x)（Φ 为标准正态 CDF） | 约 [-0.17, +inf) | 平滑版 ReLU，Transformer 中广泛使用 |
| SiLU/Swish | x * σ(x) | 约 [-0.28, +inf) | 平滑非单调，效果好 |
| Softmax | e^(x_i) / Σe^(x_j) | (0,1)，和为1 | 多分类输出层，将 logits 转为概率分布 |

### 1.4 反向传播算法（Backpropagation, BP）

**核心思想**：利用链式法则（Chain Rule）高效计算 loss 对每个参数的梯度。

**前向传播**：逐层计算输出。

$$z^{(l)} = W^{(l)} a^{(l-1)} + b^{(l)}$$
$$a^{(l)} = f(z^{(l)})$$

**反向传播**：从输出层开始，反向逐层计算梯度。

定义误差项 δ^(l) = ∂L/∂z^(l)

输出层：δ^(L) = ∂L/∂a^(L) ⊙ f'(z^(L))

隐藏层递推：δ^(l) = (W^(l+1))ᵀδ^(l+1) ⊙ f'(z^(l))

参数梯度：
- ∂L/∂W^(l) = δ^(l)(a^(l-1))ᵀ
- ∂L/∂b^(l) = δ^(l)

**计算图视角**：现代深度学习框架（PyTorch、TensorFlow）使用动态/静态计算图自动实现 Automatic Differentiation（自动微分），本质是反向传播的泛化。

### 1.5 梯度消失与梯度爆炸

**梯度消失（Vanishing Gradient）**：反向传播时梯度逐层相乘变小，导致浅层参数几乎不更新。

原因：使用 Sigmoid/Tanh（饱和区梯度接近 0）、网络层数过深。

**梯度爆炸（Exploding Gradient）**：梯度逐层相乘变大，导致参数更新过大，训练不稳定。

**解决方法**：
- ReLU 激活函数（缓解梯度消失）
- 残差连接（Residual Connection / Skip Connection）
- Batch Normalization / Layer Normalization
- 合适的权重初始化（Xavier / He 初始化）
- 梯度裁剪（Gradient Clipping）：当梯度范数超过阈值时等比缩放

**权重初始化**：
- **Xavier 初始化**：W ~ N(0, 2/(n_in + n_out))，适用于 Sigmoid/Tanh
- **He 初始化**：W ~ N(0, 2/n_in)，适用于 ReLU

## 二、卷积神经网络（CNN）

### 2.1 卷积操作

**2D 卷积**：卷积核（Filter/Kernel）在输入特征图上滑动做加权求和。

$$\text{output}(i,j) = \sum_m \sum_n \text{input}(i+m, j+n) \cdot \text{kernel}(m,n) + \text{bias}$$

关键参数：
- **卷积核大小**：常用 3x3、5x5、1x1
- **步长（Stride）**：每次滑动的距离
- **填充（Padding）**：在输入边缘补零。Same Padding 保持尺寸不变，Valid Padding 不填充
- **通道数**：输入通道 C_in，输出通道 C_out

**输出尺寸公式**：

$$O = \frac{W - K + 2P}{S} + 1$$

W：输入尺寸，K：核大小，P：Padding，S：Stride。

**参数量**：K x K x C_in x C_out + C_out（远少于全连接层）

**1x1 卷积**：改变通道数（升维/降维），跨通道信息整合，等效于对每个像素做全连接。

**深度可分离卷积（Depthwise Separable Convolution）**：先对每个通道单独卷积（Depthwise），再用 1x1 卷积混合通道（Pointwise）。参数量和计算量大幅减少，MobileNet 的核心。

### 2.2 池化（Pooling）

下采样操作，减小特征图尺寸并增强平移不变性。

- **Max Pooling**：取窗口内最大值（最常用）
- **Average Pooling**：取窗口内平均值
- **Global Average Pooling (GAP)**：将整个特征图取平均为一个值，替代全连接层

### 2.3 感受野（Receptive Field）

某一层特征图上的一个点对应输入图像上的区域大小。

- 层数越深 → 感受野越大
- 多个 3x3 卷积的堆叠：两层 3x3 等效感受野 = 5x5，三层 3x3 = 7x7，但参数更少
- 空洞卷积（Dilated Convolution / Atrous Convolution）：通过在卷积核中插入空洞来增大感受野而不增加参数

### 2.4 Batch Normalization（BN）

对每个 mini-batch 的每个通道进行归一化：

$$\hat{x}_i = \frac{x_i - \mu_B}{\sqrt{\sigma_B^2 + \epsilon}}$$
$$y_i = \gamma \hat{x}_i + \beta$$

其中 μ_B、σ²_B 为 batch 统计量，γ、β 为可学习参数。

**作用**：加速训练、允许更大学习率、一定的正则化效果、缓解梯度消失。

**测试时**：使用训练时统计的全局移动平均 μ 和 σ²。

**Layer Normalization**：对单个样本的所有特征归一化，不依赖 batch，适用于 RNN 和 Transformer。

### 2.5 经典 CNN 架构

| 网络 | 年份 | 核心贡献 |
|---|---|---|
| LeNet-5 | 1998 | 开创 CNN，手写数字识别 |
| AlexNet | 2012 | 深层 CNN + ReLU + Dropout + GPU 训练，ImageNet 冠军，开启深度学习时代 |
| VGGNet | 2014 | 统一使用 3x3 小卷积核，深度 16/19 层 |
| GoogLeNet/Inception | 2014 | Inception 模块（多尺度并行卷积 + 1x1 降维），参数高效 |
| ResNet | 2015 | **残差连接**（y = F(x) + x），解决深度退化问题，训练超 100 层网络 |
| DenseNet | 2017 | 密集连接（每层与前面所有层连接），特征复用 |
| EfficientNet | 2019 | 复合缩放（深度/宽度/分辨率同时缩放） |

**ResNet 残差连接的重要性**：
- 解决深度退化（Degradation）问题：更深的网络性能反而下降
- 恒等映射使得 F(x) = 0 就能保持原始信息，学习残差比学习完整映射更容易
- 梯度可以通过 shortcut 直接回传到浅层，缓解梯度消失
- 被后续 Transformer 广泛采用

## 三、循环神经网络（RNN）

### 3.1 基本 RNN

处理序列数据，引入隐藏状态 h_t 捕获时序信息。

$$h_t = \tanh(W_{hh} h_{t-1} + W_{xh} x_t + b_h)$$
$$y_t = W_{hy} h_t + b_y$$

**问题**：梯度消失/爆炸（长序列中梯度连乘 ∏W_{hh}），难以建模长距离依赖。

### 3.2 LSTM（Long Short-Term Memory）

通过**门控机制（Gating Mechanism）**解决长距离依赖问题。

引入**细胞状态（Cell State）** c_t 作为信息高速公路，以及三个门：

**遗忘门（Forget Gate）**：决定从 c_{t-1} 中丢弃哪些信息。

$$f_t = \sigma(W_f [h_{t-1}, x_t] + b_f)$$

**输入门（Input Gate）**：决定哪些新信息写入 c_t。

$$i_t = \sigma(W_i [h_{t-1}, x_t] + b_i)$$
$$\tilde{c}_t = \tanh(W_c [h_{t-1}, x_t] + b_c)$$

**细胞状态更新**：

$$c_t = f_t \odot c_{t-1} + i_t \odot \tilde{c}_t$$

**输出门（Output Gate）**：决定输出哪些信息。

$$o_t = \sigma(W_o [h_{t-1}, x_t] + b_o)$$
$$h_t = o_t \odot \tanh(c_t)$$

⊙ 表示逐元素乘法（Hadamard product）。Sigmoid 门输出 0~1 控制信息流通比例。

**为什么 LSTM 缓解梯度消失**：c_t = f_t ⊙ c_{t-1} + ...，当 f_t ≈ 1 时，梯度可以沿 cell state 无衰减地传播。

### 3.3 GRU（Gated Recurrent Unit）

LSTM 的简化版本，将遗忘门和输入门合并为**更新门（Update Gate）**，减少参数。

$$z_t = \sigma(W_z [h_{t-1}, x_t])$$
$$r_t = \sigma(W_r [h_{t-1}, x_t])$$
$$\tilde{h}_t = \tanh(W [r_t \odot h_{t-1}, x_t])$$
$$h_t = (1 - z_t) \odot h_{t-1} + z_t \odot \tilde{h}_t$$

z_t：更新门；r_t：重置门（Reset Gate）。GRU 参数更少，训练更快，性能与 LSTM 相当。

### 3.4 双向 RNN（Bidirectional RNN）

同时从正向和反向处理序列：

- 正向：h_t_forward = f(x_t, h_{t-1}_forward)
- 反向：h_t_backward = f(x_t, h_{t+1}_backward)
- 拼接：h_t = [h_t_forward; h_t_backward]

在 NLP 中广泛使用（如 BERT 的双向编码思想来源），但不适用于生成任务（无法看到未来）。

### 3.5 Seq2Seq（Sequence-to-Sequence）

**编码器-解码器架构**：
- **Encoder**：将输入序列编码为固定长度的上下文向量 c（最后一个隐藏状态）
- **Decoder**：根据 c 逐步生成输出序列

$$c = \text{Encoder}(x_1, ..., x_T)$$
$$y_t = \text{Decoder}(c, y_1, ..., y_{t-1})$$

**瓶颈问题**：固定长度的上下文向量难以编码长序列的全部信息 → Attention 机制的引入。

**Attention 机制（Bahdanau Attention, 2014）**：
- Decoder 每步计算对 Encoder 所有隐藏状态的注意力权重
- 上下文向量变为加权和：c_t = Σ α_ti h_i
- 注意力权重：α_ti = exp(e_ti) / Σ exp(e_tj)，e_ti = a(s_{t-1}, h_i)

## 四、Transformer

### 4.1 Self-Attention 机制

**核心思想**：序列中每个位置直接与所有其他位置交互，无需像 RNN 那样逐步传递信息。

对输入序列 X (n x d 矩阵)，通过三个线性变换得到 Query、Key、Value：

$$Q = XW^Q, \quad K = XW^K, \quad V = XW^V$$

**缩放点积注意力（Scaled Dot-Product Attention）**：

$$\text{Attention}(Q, K, V) = \text{softmax}\left(\frac{QK^T}{\sqrt{d_k}}\right) V$$

**计算过程**：
1. 计算 QKᵀ：得到 n x n 的注意力分数矩阵，衡量每对位置的关联度
2. 缩放：除以 sqrt(d_k)，防止当 d_k 较大时点积值过大导致 Softmax 梯度过小
3. Softmax：归一化为概率分布
4. 加权求和：以注意力权重对 V 加权，得到输出

**时间复杂度**：O(n²d)，n 为序列长度，这也是 Transformer 处理长序列的主要瓶颈。

### 4.2 Multi-Head Attention

将 d 维空间拆分为 h 个子空间，各自独立做注意力，再拼接。

$$\text{MultiHead}(Q, K, V) = \text{Concat}(\text{head}_1, ..., \text{head}_h) W^O$$

$$\text{head}_i = \text{Attention}(QW_i^Q, KW_i^K, VW_i^V)$$

每个 head 的维度 d_k = d/h。多头注意力使模型能从不同表示子空间学习不同类型的依赖关系（如语法依赖、语义依赖）。

典型配置：d = 512，h = 8，d_k = 64（原始 Transformer）。

### 4.3 位置编码（Positional Encoding）

Self-Attention 本身对位置无感知（排列不变性），需要额外注入位置信息。

**正弦位置编码（Sinusoidal PE）**：

$$PE_{(pos, 2i)} = \sin\left(\frac{pos}{10000^{2i/d}}\right)$$
$$PE_{(pos, 2i+1)} = \cos\left(\frac{pos}{10000^{2i/d}}\right)$$

性质：
- 每个位置的编码唯一
- 相对位置可以通过线性变换表示：PE_{pos+k} 可以表示为 PE_{pos} 的线性函数
- 可外推到训练时未见的序列长度

**其他位置编码**：
- 可学习位置编码（Learnable PE）：BERT、GPT 使用
- 旋转位置编码（RoPE, Rotary Position Embedding）：LLaMA 使用，在注意力计算中编码相对位置
- ALiBi（Attention with Linear Biases）：直接在注意力分数上加线性偏置

### 4.4 Transformer 架构

**Encoder 层**：
```
输入 → Multi-Head Self-Attention → Add & Norm → Feed-Forward Network → Add & Norm → 输出
```

**Decoder 层**：
```
输入 → Masked Multi-Head Self-Attention → Add & Norm → Cross-Attention → Add & Norm → FFN → Add & Norm → 输出
```

**关键组件**：

- **Feed-Forward Network（FFN）**：两层全连接 + 激活函数（ReLU/GELU）。FFN(x) = W₂ * GELU(W₁x + b₁) + b₂，中间层维度通常为 4d。

- **Add & Norm（残差 + LayerNorm）**：LayerNorm(x + Sublayer(x))，稳定训练。

- **Masked Self-Attention**：Decoder 中使用因果掩码（Causal Mask），防止当前位置看到未来位置的信息。将上三角注意力分数设为 -inf，Softmax 后为 0。

- **Cross-Attention**：Decoder 的 Q 来自 Decoder 自身，K、V 来自 Encoder 输出，实现 Encoder 和 Decoder 之间的信息交互。

**原始 Transformer（Vaswani et al., 2017）**：6 层 Encoder + 6 层 Decoder，d=512，h=8，FFN 中间维度 2048。

### 4.5 为什么 Transformer 优于 RNN

| 方面 | RNN | Transformer |
|---|---|---|
| 并行化 | 时间步顺序依赖，无法并行 | Self-Attention 可完全并行计算 |
| 长距离依赖 | 信息需逐步传递，距离越远衰减越大 | 任意两个位置直接交互，路径长度 O(1) |
| 训练效率 | O(n) 的顺序计算 | O(1) 的并行深度（但总计算量 O(n²d)） |
| 梯度传播 | 链式乘法导致梯度消失/爆炸 | 残差连接 + 注意力的直接路径 |
| 可扩展性 | 难以扩展到非常大的模型 | 架构简洁，高度可扩展（Scaling Law） |

## 五、生成模型

### 5.1 GAN（Generative Adversarial Network）

**核心思想**：生成器 G 和判别器 D 进行对抗博弈。

- **Generator** G(z)：从随机噪声 z ~ p_z 生成假样本
- **Discriminator** D(x)：判断样本是真实的还是生成的

**目标函数**（Minimax Game）：

$$\min_G \max_D \mathbb{E}_{x \sim p_{data}}[\log D(x)] + \mathbb{E}_{z \sim p_z}[\log(1 - D(G(z)))]$$

**训练过程**：
1. 固定 G，训练 D 最大化判别能力
2. 固定 D，训练 G 最小化 D 对假样本的识别能力
3. 交替进行直至达到纳什均衡（Nash Equilibrium）：p_G = p_data

**训练难点**：
- **模式崩塌（Mode Collapse）**：G 只生成有限的几种样本
- **训练不稳定**：G 和 D 需要保持平衡
- **评估困难**：缺乏直接的训练目标来衡量生成质量

**改进**：WGAN（使用 Wasserstein 距离）、StyleGAN（高质量图像生成）、Progressive GAN 等。

### 5.2 VAE（Variational Autoencoder）

**目标**：学习数据的潜在分布 p(z|x)，从中采样生成新样本。

**架构**：
- **Encoder** q_φ(z|x)：将输入编码为潜在空间的分布参数（μ, σ²）
- **Decoder** p_θ(x|z)：从潜在变量重构数据

**Loss Function（ELBO，Evidence Lower Bound 的负值）**：

$$\mathcal{L} = -\mathbb{E}_{q_\phi(z|x)}[\log p_\theta(x|z)] + D_{KL}(q_\phi(z|x) \| p(z))$$

- **重构损失**：-E[log p_θ(x|z)]，确保解码器能重构输入
- **KL 散度正则化**：D_KL(q_φ(z|x) ‖ p(z))，使后验分布接近先验 p(z) = N(0, I)

**重参数化技巧（Reparameterization Trick）**：z = μ + σ ⊙ ε，ε ~ N(0, I)，使采样过程可导。

### 5.3 Diffusion Model（扩散模型）

**基本原理**：

**前向过程（Forward Process / Diffusion）**：逐步向数据添加高斯噪声，直至变为纯噪声。

$$q(x_t | x_{t-1}) = \mathcal{N}(x_t; \sqrt{1 - \beta_t} x_{t-1}, \beta_t I)$$

经过 T 步后 x_T 约等于 N(0, I)。

可直接从 x₀ 得到任意步的 x_t：q(x_t | x₀) = N(x_t; sqrt(α_bar_t) x₀, (1-α_bar_t)I)，其中 α_bar_t = ∏(1-β_s)。

**反向过程（Reverse Process / Denoising）**：学习从噪声逐步去噪还原数据。

$$p_\theta(x_{t-1}|x_t) = \mathcal{N}(x_{t-1}; \mu_\theta(x_t, t), \Sigma_\theta(x_t, t))$$

**训练目标**：训练一个神经网络 ε_θ 预测每步加入的噪声。

$$L = \mathbb{E}_{t, x_0, \epsilon}[\|\epsilon - \epsilon_\theta(x_t, t)\|^2]$$

**采样过程**：从 x_T ~ N(0, I) 开始，用训练好的 ε_θ 逐步去噪得到 x₀。

**优点**：训练稳定（不像 GAN）、生成质量高、理论基础扎实。
**缺点**：采样慢（需要多步去噪），可通过 DDIM、Latent Diffusion 等方法加速。
