# 概率统计与线性代数

这部分常用于机器学习、算法分析、随机过程和数据处理的基础题。笔试通常不会要求很深推导，但会考概念、公式含义和简单计算。

---

## 一、概率论基础

### 1.1 基本概念

- **样本空间**：所有可能结果的集合。
- **事件**：样本空间的子集。
- **互斥事件**：不能同时发生。
- **独立事件**：一个事件发生不影响另一个事件发生。

重要公式：

```text
P(A ∪ B) = P(A) + P(B) - P(A ∩ B)
P(A | B) = P(A ∩ B) / P(B)
若 A, B 独立，则 P(A ∩ B) = P(A)P(B)
```

### 1.2 全概率公式与贝叶斯公式

若 `{B1, B2, ..., Bn}` 构成样本空间划分：

```text
P(A) = Σ P(A | Bi)P(Bi)
P(Bi | A) = P(A | Bi)P(Bi) / P(A)
```

贝叶斯公式常用于朴素贝叶斯分类、医学检测题、垃圾邮件过滤等场景。

## 二、随机变量与分布

### 2.1 随机变量

- **离散型随机变量**：取值可数，如伯努利分布、二项分布、泊松分布。
- **连续型随机变量**：取值连续，如均匀分布、正态分布、指数分布。

### 2.2 常见分布

| 分布 | 记号 | 期望 | 方差 | 场景 |
|---|---|---|---|---|
| 伯努利分布 | Bernoulli(p) | p | p(1-p) | 一次成功/失败 |
| 二项分布 | B(n,p) | np | np(1-p) | n 次独立伯努利试验 |
| 泊松分布 | Poisson(λ) | λ | λ | 单位时间事件次数 |
| 均匀分布 | U(a,b) | (a+b)/2 | (b-a)^2/12 | 区间内等可能 |
| 正态分布 | N(μ,σ²) | μ | σ² | 自然测量误差 |
| 指数分布 | Exp(λ) | 1/λ | 1/λ² | 等待时间 |

### 2.3 期望、方差与协方差

```text
E(X) = Σ xP(X=x) 或 ∫ xf(x)dx
Var(X) = E[(X-E(X))^2] = E(X^2) - [E(X)]^2
Cov(X,Y) = E[(X-E(X))(Y-E(Y))]
```

- 方差衡量离散程度。
- 协方差衡量两个变量共同变化方向。
- 相关系数是归一化后的协方差，范围为 `[-1, 1]`。

### 2.4 大数定律与中心极限定理

- **大数定律**：样本均值随样本数增大趋近真实期望。
- **中心极限定理**：大量独立同分布随机变量之和近似服从正态分布。

这两个定理解释了为什么抽样估计可行，以及为什么正态分布极其常见。

## 三、统计基础

### 3.1 参数估计

- **点估计**：用一个数估计参数，如样本均值估计总体均值。
- **区间估计**：给出置信区间，如 95% 置信区间表示重复抽样 100 次约 95 次包含真值。
- **无偏估计**：估计量期望等于真实参数。样本均值是 μ 的无偏估计，样本方差除以 n-1 是 σ² 的无偏估计。
- **一致估计（相合估计）**：样本数增大时估计量依概率收敛到真实参数。
- **有效估计**：在无偏估计中方差最小的。Cramér-Rao 下界给出方差的下限。

**贝叶斯估计 vs 频率学派估计**：

| 对比 | 频率学派 | 贝叶斯学派 |
|------|----------|-----------|
| 参数观 | 参数是固定但未知的常数 | 参数是随机变量，有先验分布 |
| 核心方法 | MLE、矩估计 | 后验分布 = 先验 × 似然 / 归一化 |
| 结果 | 点估计 + 置信区间 | 后验分布 → MAP 估计或后验均值 |
| 优点 | 客观，不依赖先验选择 | 可融入先验知识，小样本更稳健 |

**共轭先验**：如果先验和后验属于同一分布族，则称先验为似然函数的共轭先验。例如：
- 二项分布的共轭先验是 Beta 分布
- 正态分布（已知方差）的共轭先验是正态分布
- 泊松分布的共轭先验是 Gamma 分布

### 3.2 最大似然估计 MLE

最大似然估计选择让观测数据出现概率最大的参数：

```text
θ_hat = argmax L(θ) = argmax P(data | θ)
```

实际计算常对似然取对数，变成 log-likelihood，乘积转求和，便于求导和数值稳定。

**MAP（最大后验估计）**：在 MLE 基础上加入先验分布。

```text
θ_MAP = argmax P(θ | data) = argmax P(data | θ) · P(θ)
```

当先验为均匀分布时 MAP 退化为 MLE。L2 正则化等价于高斯先验下的 MAP。

### 3.3 假设检验

- **原假设 H0**：默认成立的假设。
- **备择假设 H1**：希望证明的假设。
- **p-value**：在 H0 成立时，观察到当前或更极端结果的概率。
- **显著性水平 α**：拒绝 H0 的阈值，如 0.05。

常见错误：

- 一类错误（α 错误）：H0 为真却拒绝（假阳性）。
- 二类错误（β 错误）：H0 为假却不拒绝（假阴性）。
- 检验功效 = 1 - β，即正确拒绝错误 H0 的概率。

### 3.4 常用检验方法

| 检验 | 用途 | 条件 |
|------|------|------|
| **z 检验** | 单样本均值检验 | 总体方差已知或大样本（n ≥ 30） |
| **t 检验** | 单样本/双样本均值检验 | 小样本、总体方差未知、近似正态 |
| **配对 t 检验** | 配对样本差值的均值检验 | 前后对比实验 |
| **卡方检验 (χ²)** | 拟合优度检验、独立性检验 | 分类数据 |
| **F 检验** | 两个总体方差是否相等 | 方差分析 ANOVA 的基础 |
| **ANOVA (方差分析)** | 多个组均值是否有差异 | 多组比较 |
| **Kolmogorov-Smirnov** | 样本是否服从某分布 | 连续分布的拟合优度 |

**多重检验问题**：同时进行多次假设检验时，整体犯一类错误的概率增大。校正方法：Bonferroni 校正（α/m）、FDR 控制（Benjamini-Hochberg）。

### 3.5 采样方法

| 方法 | 思想 | 应用 |
|------|------|------|
| **简单随机采样** | 等概率抽取 | 基础 |
| **分层采样** | 按属性分层，层内随机采样 | 保证各子群体比例 |
| **重要性采样** | 用提议分布 q(x) 替代目标分布 p(x) 采样，用权重 w(x) = p(x)/q(x) 修正 | 稀有事件模拟、变分推断 |
| **MCMC** | 构造马尔可夫链，使其平稳分布为目标分布 | 贝叶斯推断 |
| **Metropolis-Hastings** | MCMC 的通用方法，用接受-拒绝准则 | 任意分布 |
| **Gibbs 采样** | MCMC 的特例，逐维条件采样 | 多维分布、LDA 主题模型 |

## 四、线性代数基础

### 4.1 向量与矩阵

- 向量可表示特征、方向或状态。
- 矩阵可表示线性变换、图邻接关系、数据集。
- 矩阵乘法本质是线性变换复合。

### 4.2 秩、线性相关与线性方程组

- **线性相关**：某个向量能由其他向量线性表示。
- **矩阵的秩**：最大线性无关行/列数量。
- 方程组 `Ax=b` 是否有解与 `rank(A)`、`rank([A|b])` 有关。

### 4.3 特征值与特征向量

```text
Av = λv
```

含义：矩阵作用在特征向量上只改变长度，不改变方向。常用于 PCA、PageRank、稳定性分析。

**计算**：解特征方程 det(A - λI) = 0 得特征值 λ，再解 (A - λI)v = 0 得特征向量。

**性质**：
- 特征值之和 = 迹 trace(A)
- 特征值之积 = det(A)
- 实对称矩阵的特征值全为实数，特征向量正交
- 相似矩阵有相同特征值

**谱分解（特征分解）**：实对称矩阵 A = QΛQ^T，其中 Q 为正交矩阵（列为特征向量），Λ 为特征值对角矩阵。

### 4.4 正定矩阵

实对称矩阵 A 是正定的，当且仅当以下等价条件之一成立：
- 对所有非零向量 x，x^T Ax > 0
- 所有特征值 > 0
- 所有顺序主子式 > 0
- 存在可逆矩阵 L 使得 A = L^T L（Cholesky 分解）

半正定：将 > 换为 ≥。

**应用**：Hessian 矩阵正定 → 该点为严格局部最小值。协方差矩阵是半正定的。

### 4.5 正交、投影与最小二乘

- **正交**：内积为 0。
- **Gram-Schmidt 正交化**：将一组线性无关向量转化为正交（正规正交）基。
- **投影**：把向量映射到子空间中最接近的位置。投影矩阵 P = A(A^T A)^{-1} A^T。
- **最小二乘**：最小化残差平方和 `||Ax-b||^2`，常用于线性回归。

正规方程：

```text
X^T X w = X^T y
```

实际工程中常用 QR 分解、SVD 或梯度法，避免直接求逆带来的数值不稳定。

### 4.6 矩阵分解

| 分解 | 形式 | 用途 |
|---|---|---|
| LU 分解 | A = LU（下三角 × 上三角） | 解线性方程组（高斯消元的矩阵形式） |
| Cholesky 分解 | A = LL^T（正定矩阵） | 比 LU 快一倍，协方差矩阵处理 |
| QR 分解 | A = QR（正交 × 上三角） | 最小二乘、特征值计算 |
| 特征分解 | A = QΛQ^{-1} | 对角化、谱分析、PCA |
| SVD | A = UΣV^T | 降维、推荐系统、低秩近似、伪逆 |

**SVD 详解**：任意 m×n 矩阵 A 都可以分解为 A = UΣV^T：
- U (m×m)：左奇异向量矩阵（AA^T 的特征向量）
- Σ (m×n)：奇异值对角矩阵（σ₁ ≥ σ₂ ≥ ... ≥ 0）
- V (n×n)：右奇异向量矩阵（A^T A 的特征向量）

截断 SVD（取前 k 个奇异值）给出最优的秩-k 近似（Eckart-Young 定理）。

### 4.7 矩阵求导

矩阵求导在机器学习中非常常用（梯度下降的基础）。

**常用公式**：

```text
∂(a^T x) / ∂x = a
∂(x^T Ax) / ∂x = (A + A^T)x    （若 A 对称则 = 2Ax）
∂(||Ax - b||²) / ∂x = 2A^T(Ax - b)
∂ tr(AB) / ∂A = B^T
```

**链式法则**：对标量函数 f = f(g(x))，∂f/∂x = (∂f/∂g)(∂g/∂x)。矩阵版本需要注意维度和转置的匹配。

## 五、优化基础

### 5.1 梯度与 Hessian

- **梯度** ∇f(x)：函数上升最快方向，是各偏导数组成的向量。
- **Hessian** H(x)：二阶偏导数组成的矩阵，描述曲率。H 正定 → 严格局部最小值；H 负定 → 严格局部最大值。
- **梯度下降**：x_{k+1} = x_k - α∇f(x_k)，沿负梯度方向更新参数。α 为学习率。

**梯度下降的变种**：
- **批量梯度下降 (BGD)**：用全部数据计算梯度，稳定但慢
- **随机梯度下降 (SGD)**：每次用一个样本，快但噪声大
- **小批量 SGD (Mini-batch)**：折中，通常 batch size = 32~256
- **动量 (Momentum)**：引入历史梯度的指数加权平均，加速收敛
- **Adam**：结合动量和自适应学习率（RMSProp），最常用的优化器

### 5.2 凸优化

- **凸集**：集合中任意两点连线仍在集合内。
- **凸函数**：f(λx + (1-λ)y) ≤ λf(x) + (1-λ)f(y)。等价于 Hessian 半正定。
- **凸优化的核心性质**：任意局部最优都是全局最优。
- 常见凸损失：线性回归的 MSE、逻辑回归的交叉熵。

### 5.3 约束优化

**拉格朗日乘子法**：将等式约束优化问题转化为无约束问题。

```text
min f(x)  s.t. g(x) = 0
→ L(x, λ) = f(x) + λg(x)
→ 解 ∇L = 0
```

**KKT 条件 (Karush-Kuhn-Tucker)**：处理不等式约束的必要条件。

对问题 min f(x) s.t. g_i(x) ≤ 0, h_j(x) = 0：

1. **稳定性**：∇f + Σμ_i ∇g_i + Σλ_j ∇h_j = 0
2. **原始可行性**：g_i(x) ≤ 0, h_j(x) = 0
3. **对偶可行性**：μ_i ≥ 0
4. **互补松弛**：μ_i · g_i(x) = 0（约束要么取等，要么乘子为零）

**应用**：SVM 的最优间隔分类器就是通过 KKT 条件推导的。

### 5.4 信息论基础

| 概念 | 定义 | 含义 |
|------|------|------|
| 熵 H(X) | -Σ p(x) log p(x) | X 的不确定性（平均信息量） |
| 条件熵 H(Y\|X) | -Σ p(x,y) log p(y\|x) | 已知 X 后 Y 的剩余不确定性 |
| 互信息 I(X;Y) | H(X) - H(X\|Y) | X 和 Y 共享的信息量 |
| KL 散度 D_KL(P\|\|Q) | Σ p(x) log(p(x)/q(x)) | P 和 Q 之间的"距离"（不对称） |
| 交叉熵 H(P,Q) | -Σ p(x) log q(x) | 用 Q 编码 P 的平均编码长度 |

关系：H(P,Q) = H(P) + D_KL(P||Q)。最小化交叉熵等价于最小化 KL 散度（因为 H(P) 是常数）。

交叉熵是分类任务最常用的损失函数。
