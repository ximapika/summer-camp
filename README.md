# 北京大学计算机夏令营复习知识库

本知识库为北京大学优秀大学生夏令营准备，涵盖**笔试**、**面试**、**上机**三大环节，覆盖计算机科学核心知识体系。笔试部分按“计算机学生应该掌握的常识”组织，既包含考研/本科核心课，也包含 Linux、开发工具、AI、分布式系统等容易被问到的工程常识。

## 使用建议

### 笔试复习
按照知识体系系统复习。**第一优先级**：操作系统、计算机网络、数据结构、数据库、计算机组成原理、Linux 基础。**第二优先级**：编译原理、离散数学、软件工程、信息安全、程序设计语言、开发工具。**加分/开放题方向**：人工智能与机器学习、分布式系统、概率统计与线性代数。每个文件内容自成体系，可以独立阅读。

### 面试准备
1. 先准备好**自我介绍**（1-3 分钟版本），熟练描述项目经历
2. 重点背诵**计算机基础问答**中的高频题
3. 了解**前沿技术**，能对 LLM、分布式系统等话题有自己的见解
4. 对自己的项目准备“原理、难点、替代方案、实验/指标、失败教训”五类追问

### 上机备战
1. 先掌握 **C++ STL** 的常用容器和算法（sort/lower_bound/priority_queue 等）
2. 按照 **基础算法 → 搜索 → 动态规划 → 图论** 的顺序递进复习
3. 每个算法模板都用 C++ 编写，可直接用于上机

## 内容说明

- 语言：中文为主，专业术语保留英文（如 TCP、Cache、Transformer）
- 上机部分所有代码模板均为 **C++**，基于 C++17 标准
- 每个知识点包含：概念解释、核心要点、对比表格、代码示例（上机部分）
- “知识清单.md”适合考前快速扫盲；各专题文件适合逐章复习和查漏补缺
