# 搜索算法

## 一、深度优先搜索 (DFS)

### 1.1 概念

DFS（Depth-First Search）是一种"一条路走到黑"的搜索策略。从起点出发，沿一条路径尽可能深入探索，直到无法继续时回退（backtrack），再尝试其他路径。DFS 本质上是对搜索树的前序遍历。

### 1.2 递归实现

```cpp
bool visited[N];
void dfs(int u) {
    visited[u] = true;
    // 处理节点 u
    for (int v : adj[u]) {
        if (!visited[v]) {
            dfs(v);
        }
    }
}
```

### 1.3 回溯法 (Backtracking)

回溯法是 DFS 的一种应用方式：在搜索过程中，当发现当前状态无法得到可行解时，撤销上一步操作（"回溯"），尝试其他选择。核心是"选择 → 递归 → 撤销选择"。

```cpp
void backtrack(vector<int>& path, vector<bool>& used) {
    if (path.size() == n) {
        // 找到一个解，记录
        result.push_back(path);
        return;
    }
    for (int i = 0; i < n; i++) {
        if (used[i]) continue;
        // 剪枝条件（可选）
        if (should_prune(i)) continue;

        path.push_back(i);    // 选择
        used[i] = true;
        backtrack(path, used); // 递归
        path.pop_back();       // 撤销选择
        used[i] = false;
    }
}
```

### 1.4 剪枝优化

剪枝是指在搜索过程中，提前判断当前分支不可能产生更优解或可行解，直接跳过该分支，从而大幅减少搜索量。

常见剪枝策略：
- **可行性剪枝**：当前状态已经违反约束，不再继续
- **最优性剪枝**：当前解已经不可能比已知最优解更好
- **对称性剪枝**：利用问题的对称性避免重复搜索
- **排序剪枝**：将选择排序后搜索，更早触发其他剪枝

## 二、广度优先搜索 (BFS)

### 2.1 概念

BFS（Breadth-First Search）是"逐层扩展"的搜索策略。从起点出发，先访问所有距离为 1 的节点，再访问距离为 2 的节点，以此类推。BFS 使用队列实现，天然求出的是**最短路径**（在无权图中）。

### 2.2 模板

```cpp
int bfs(int start, int target) {
    queue<int> q;
    unordered_set<int> visited;
    q.push(start);
    visited.insert(start);
    int step = 0;
    while (!q.empty()) {
        int sz = q.size();
        for (int i = 0; i < sz; i++) {
            int cur = q.front(); q.pop();
            if (cur == target) return step;
            for (int next : get_neighbors(cur)) {
                if (!visited.count(next)) {
                    visited.insert(next);
                    q.push(next);
                }
            }
        }
        step++;
    }
    return -1;  // 不可达
}
```

### 2.3 多源 BFS

多源 BFS 是指同时从多个起点出发进行 BFS。做法是将所有起点同时入队，然后正常 BFS 即可。

**经典例题**：矩阵中求每个位置到最近的 0 的距离 → 将所有 0 作为起点同时 BFS。

```cpp
// 将所有起点入队
for (所有起点 s) {
    q.push(s);
    dist[s] = 0;
}
// 正常 BFS
while (!q.empty()) { ... }
```

## 三、经典搜索问题

### 3.1 全排列

给定 n 个不同的数，生成所有排列。使用回溯法，每次从未使用的数中选一个。

```cpp
void permute(vector<int>& nums, int idx, vector<vector<int>>& res) {
    if (idx == nums.size()) {
        res.push_back(nums);
        return;
    }
    for (int i = idx; i < nums.size(); i++) {
        swap(nums[idx], nums[i]);
        permute(nums, idx + 1, res);
        swap(nums[idx], nums[i]);
    }
}
```

也可以用 STL 的 `next_permutation`：

```cpp
sort(nums.begin(), nums.end());
do {
    // 处理当前排列
} while (next_permutation(nums.begin(), nums.end()));
```

### 3.2 N 皇后

在 N×N 棋盘上放 N 个皇后，使得任意两个皇后不在同一行、同一列、同一对角线上。

```cpp
int n;
bool col[N], dg[2*N], udg[2*N];  // 列、主对角线、副对角线

void solve(int row) {
    if (row == n) {
        ans++;
        return;
    }
    for (int c = 0; c < n; c++) {
        if (!col[c] && !dg[row - c + n] && !udg[row + c]) {
            col[c] = dg[row - c + n] = udg[row + c] = true;
            solve(row + 1);
            col[c] = dg[row - c + n] = udg[row + c] = false;
        }
    }
}
```

对角线编码技巧：主对角线上 row-col 相同，副对角线上 row+col 相同。

### 3.3 数独

9×9 棋盘填数 1-9，每行、每列、每个 3×3 宫内数字不重复。典型的约束满足问题（CSP），用回溯法解决。

优化：优先填可选数字最少的空格（MRV 启发式）。

### 3.4 迷宫问题

从起点到终点的最短路径 → BFS。四方向移动：

```cpp
int dx[] = {-1, 1, 0, 0};
int dy[] = {0, 0, -1, 1};

int bfs_maze(vector<vector<int>>& grid, int sx, int sy, int ex, int ey) {
    int n = grid.size(), m = grid[0].size();
    queue<pair<int,int>> q;
    vector<vector<int>> dist(n, vector<int>(m, -1));
    q.push({sx, sy});
    dist[sx][sy] = 0;
    while (!q.empty()) {
        auto [x, y] = q.front(); q.pop();
        if (x == ex && y == ey) return dist[x][y];
        for (int d = 0; d < 4; d++) {
            int nx = x + dx[d], ny = y + dy[d];
            if (nx >= 0 && nx < n && ny >= 0 && ny < m
                && grid[nx][ny] == 0 && dist[nx][ny] == -1) {
                dist[nx][ny] = dist[x][y] + 1;
                q.push({nx, ny});
            }
        }
    }
    return -1;
}
```

## 四、记忆化搜索

### 4.1 概念

记忆化搜索是在 DFS 的基础上，用一个表（通常是数组或 map）记录已经计算过的状态的结果。当再次遇到相同状态时，直接返回之前的结果，避免重复计算。本质上是**自顶向下的动态规划**。

### 4.2 示例：斐波那契

```cpp
int memo[N];
int fib(int n) {
    if (n <= 1) return n;
    if (memo[n] != -1) return memo[n];
    return memo[n] = fib(n-1) + fib(n-2);
}
// 初始化：memset(memo, -1, sizeof memo);
```

### 4.3 与 DP 的关系

记忆化搜索和动态规划解决的是同一类问题（有重叠子问题和最优子结构），只是方向不同：
- **记忆化搜索**：自顶向下（从大问题递归到小问题）
- **DP**：自底向上（从小问题迭代到大问题）

有些问题（如树形结构上的问题）用记忆化搜索更自然。

## 五、迭代加深搜索 (IDA*)

### 5.1 概念

迭代加深搜索（Iterative Deepening DFS, IDDFS）结合了 BFS 的完备性和 DFS 的低空间消耗。做法是：设定一个深度限制 maxDepth，在此限制内进行 DFS；若未找到解，增大 maxDepth 重试。

IDA* 在此基础上加入**估价函数** h()：如果当前深度 + h(当前状态) > maxDepth，则剪枝。

```cpp
int maxDepth;

bool dfs(int depth, State state) {
    if (depth + h(state) > maxDepth) return false;  // IDA* 剪枝
    if (is_goal(state)) return true;
    for (auto next : get_moves(state)) {
        if (dfs(depth + 1, next)) return true;
    }
    return false;
}

int ida_star(State start) {
    for (maxDepth = h(start); ; maxDepth++) {
        if (dfs(0, start)) return maxDepth;
    }
}
```

### 5.2 适用场景

当搜索树很深但解在较浅处，且 BFS 的队列会爆内存时，IDA* 是好选择。空间复杂度只有 O(深度)。

## 六、双向 BFS

### 6.1 概念

从起点和终点同时进行 BFS，当两个方向的搜索"相遇"时，路径即被找到。相比单向 BFS，搜索的节点数从 O(b^d) 降低到 O(b^(d/2))，其中 b 是分支因子，d 是深度。

### 6.2 模板

```cpp
int bidirectional_bfs(State start, State target) {
    unordered_map<State, int> dist_s, dist_t;
    queue<State> qs, qt;
    qs.push(start); dist_s[start] = 0;
    qt.push(target); dist_t[target] = 0;

    while (!qs.empty() && !qt.empty()) {
        int res = -1;
        // 从较小的一端扩展
        if (qs.size() <= qt.size())
            res = expand(qs, dist_s, dist_t);
        else
            res = expand(qt, dist_t, dist_s);
        if (res != -1) return res;
    }
    return -1;
}

int expand(queue<State>& q, unordered_map<State,int>& dist_cur,
           unordered_map<State,int>& dist_other) {
    int sz = q.size();
    for (int i = 0; i < sz; i++) {
        State cur = q.front(); q.pop();
        for (auto next : get_neighbors(cur)) {
            if (dist_cur.count(next)) continue;
            dist_cur[next] = dist_cur[cur] + 1;
            if (dist_other.count(next))
                return dist_cur[next] + dist_other[next];
            q.push(next);
        }
    }
    return -1;
}
```

## 七、A* 算法

### 7.1 概念

A* 是带有启发式的最短路径算法，结合了 Dijkstra 的最优性和贪心搜索的速度。对每个状态维护：
- g(n)：从起点到 n 的实际代价
- h(n)：从 n 到终点的**估计**代价（启发函数）
- f(n) = g(n) + h(n)：总估计代价

每次从优先队列中取出 f 值最小的节点扩展。

### 7.2 要求

h(n) 必须满足**可采纳性 (admissible)**：h(n) ≤ 实际最短距离（不高估），这样 A* 保证找到最优解。

常用启发函数：
- 网格图：曼哈顿距离（四方向移动）、欧氏距离
- 八数码：不在位的数字个数 / 曼哈顿距离之和

### 7.3 模板

```cpp
struct Node {
    int state, g, f;
    bool operator>(const Node& o) const { return f > o.f; }
};

int astar(int start, int target) {
    priority_queue<Node, vector<Node>, greater<Node>> pq;
    unordered_map<int, int> dist;
    pq.push({start, 0, h(start)});
    dist[start] = 0;

    while (!pq.empty()) {
        auto [state, g, f] = pq.top(); pq.pop();
        if (state == target) return g;
        if (g > dist[state]) continue;  // 已有更优路径
        for (auto [next, w] : get_neighbors(state)) {
            int ng = g + w;
            if (!dist.count(next) || ng < dist[next]) {
                dist[next] = ng;
                pq.push({next, ng, ng + h(next)});
            }
        }
    }
    return -1;
}
```

### 7.4 经典例题：八数码

3×3 棋盘上有 1-8 和一个空格，通过移动空格将棋盘恢复为目标状态。

- 状态表示：将棋盘压缩为字符串或整数
- 启发函数：所有数字到目标位置的曼哈顿距离之和
- 判断有无解：统计逆序对数，偶数有解

```
初始状态        目标状态
2 8 3          1 2 3
1 6 4    →     4 5 6
7 0 5          7 8 0
```

## 八、经典例题精讲

### 8.1 LC 200 - 岛屿数量

给定一个由 `'1'`（陆地）和 `'0'`（水）组成的二维网格，计算岛屿的数量。岛屿被水包围，且由水平或垂直方向上相邻的陆地连接形成。

**思路**：遍历整个网格，每遇到一个 `'1'`，就用 BFS/DFS 将与之相连的所有陆地标记为 `'0'`（表示已访问），同时岛屿计数 +1。

```cpp
class Solution {
public:
    int numIslands(vector<vector<char>>& grid) {
        int m = grid.size(), n = grid[0].size();
        int count = 0;
        for (int i = 0; i < m; i++) {
            for (int j = 0; j < n; j++) {
                if (grid[i][j] == '1') {
                    count++;
                    dfs(grid, i, j, m, n);
                }
            }
        }
        return count;
    }

    void dfs(vector<vector<char>>& grid, int x, int y, int m, int n) {
        if (x < 0 || x >= m || y < 0 || y >= n || grid[x][y] == '0')
            return;
        grid[x][y] = '0';  // 标记为已访问
        dfs(grid, x + 1, y, m, n);
        dfs(grid, x - 1, y, m, n);
        dfs(grid, x, y + 1, m, n);
        dfs(grid, x, y - 1, m, n);
    }
};
```

### 8.2 LC 79 - 单词搜索

给定一个二维字符网格和一个单词，判断网格中是否存在该单词。单词可以由上下左右相邻的格子中的字母组成，同一格子在一个单词中不能重复使用。

**思路**：从网格中每个位置出发尝试 DFS 匹配。为避免同一路径中重复访问某个格子，进入时将其临时改为特殊字符，回溯时恢复原值。

```cpp
class Solution {
public:
    int dx[4] = {0, 0, 1, -1};
    int dy[4] = {1, -1, 0, 0};

    bool exist(vector<vector<char>>& board, string word) {
        int m = board.size(), n = board[0].size();
        for (int i = 0; i < m; i++) {
            for (int j = 0; j < n; j++) {
                if (dfs(board, word, i, j, 0))
                    return true;
            }
        }
        return false;
    }

    bool dfs(vector<vector<char>>& board, const string& word,
             int x, int y, int idx) {
        if (idx == word.size()) return true;
        int m = board.size(), n = board[0].size();
        if (x < 0 || x >= m || y < 0 || y >= n || board[x][y] != word[idx])
            return false;

        char tmp = board[x][y];
        board[x][y] = '#';  // 临时标记，防止重复访问

        for (int d = 0; d < 4; d++) {
            if (dfs(board, word, x + dx[d], y + dy[d], idx + 1))
                return true;
        }

        board[x][y] = tmp;  // 回溯恢复
        return false;
    }
};
```

### 8.3 LC 39 - 组合总和

给定一个无重复元素的整数数组 `candidates` 和一个目标值 `target`，找出所有和为 `target` 的组合。同一个数字可以被无限次选取。

**思路**：回溯搜索。每次递归从当前下标 `start` 开始选择（而不是从 0 开始），这样既允许重复选取当前数字，又避免产生重复组合。当剩余目标值为 0 时收集结果，小于 0 时剪枝返回。

```cpp
class Solution {
public:
    vector<vector<int>> combinationSum(vector<int>& candidates, int target) {
        vector<vector<int>> res;
        vector<int> path;
        sort(candidates.begin(), candidates.end());  // 排序便于剪枝
        backtrack(candidates, target, 0, path, res);
        return res;
    }

    void backtrack(vector<int>& candidates, int remain, int start,
                   vector<int>& path, vector<vector<int>>& res) {
        if (remain == 0) {
            res.push_back(path);
            return;
        }
        for (int i = start; i < candidates.size(); i++) {
            if (candidates[i] > remain) break;  // 剪枝：后面更大，不可能
            path.push_back(candidates[i]);
            // 不用 i+1，因为同一数字可重复使用
            backtrack(candidates, remain - candidates[i], i, path, res);
            path.pop_back();  // 撤销选择
        }
    }
};
```

### 8.4 LC 752 - 打开转盘锁

转盘锁有 4 个拨轮，每个拨轮上有数字 0-9，初始状态为 `"0000"`。给定一组"死亡数字"（不能经过的状态），求从 `"0000"` 转到目标 `target` 的最少旋转次数。每步可以将某一个拨轮向上或向下拨动一位。

**思路**：典型的 BFS 最短路问题。状态空间为所有 4 位数字串，每个状态有 8 个邻居（4 个轮 × 正反两个方向）。用 `unordered_set` 存死亡数字和已访问状态，逐层扩展直到找到 target。

```cpp
class Solution {
public:
    int openLock(vector<string>& deadends, string target) {
        unordered_set<string> dead(deadends.begin(), deadends.end());
        if (dead.count("0000")) return -1;
        if (target == "0000") return 0;

        unordered_set<string> visited;
        queue<string> q;
        q.push("0000");
        visited.insert("0000");
        int step = 0;

        while (!q.empty()) {
            int sz = q.size();
            step++;
            for (int i = 0; i < sz; i++) {
                string cur = q.front(); q.pop();
                // 每个拨轮可以向上或向下拨，共 8 种选择
                for (int j = 0; j < 4; j++) {
                    for (int d = -1; d <= 1; d += 2) {
                        string next = cur;
                        next[j] = (cur[j] - '0' + d + 10) % 10 + '0';
                        if (next == target) return step;
                        if (!dead.count(next) && !visited.count(next)) {
                            visited.insert(next);
                            q.push(next);
                        }
                    }
                }
            }
        }
        return -1;  // 不可达
    }
};
```
