# 基础算法

## 一、排序

### 1.1 快速排序

核心思想：选取 pivot，将数组划分为 ≤pivot 和 ≥pivot 两部分，递归排序。

时间复杂度：平均 O(n log n)，最坏 O(n²)（有序时）。空间 O(log n)。

```cpp
void quick_sort(int a[], int l, int r) {
    if (l >= r) return;
    int x = a[(l + r) >> 1], i = l - 1, j = r + 1;
    while (i < j) {
        do i++; while (a[i] < x);
        do j--; while (a[j] > x);
        if (i < j) swap(a[i], a[j]);
    }
    quick_sort(a, l, j);
    quick_sort(a, j + 1, r);
}
```

### 1.2 归并排序

核心思想：分治，将数组一分为二，分别排序后合并。

时间复杂度：O(n log n)，稳定排序。空间 O(n)。

```cpp
int tmp[N];
void merge_sort(int a[], int l, int r) {
    if (l >= r) return;
    int mid = (l + r) >> 1;
    merge_sort(a, l, mid);
    merge_sort(a, mid + 1, r);
    int k = 0, i = l, j = mid + 1;
    while (i <= mid && j <= r) {
        if (a[i] <= a[j]) tmp[k++] = a[i++];
        else tmp[k++] = a[j++];
    }
    while (i <= mid) tmp[k++] = a[i++];
    while (j <= r) tmp[k++] = a[j++];
    for (i = l, j = 0; i <= r; i++, j++) a[i] = tmp[j];
}
```

### 1.3 堆排序

时间复杂度：O(n log n)，原地排序，不稳定。

```cpp
void heap_sort(int a[], int n) {
    // 建堆
    for (int i = n / 2; i >= 1; i--) {
        int k = i;
        while (2 * k <= n) {
            int j = 2 * k;
            if (j + 1 <= n && a[j + 1] > a[j]) j++;
            if (a[k] >= a[j]) break;
            swap(a[k], a[j]);
            k = j;
        }
    }
    // 排序
    for (int i = n; i > 1; i--) {
        swap(a[1], a[i]);
        int k = 1, sz = i - 1;
        while (2 * k <= sz) {
            int j = 2 * k;
            if (j + 1 <= sz && a[j + 1] > a[j]) j++;
            if (a[k] >= a[j]) break;
            swap(a[k], a[j]);
            k = j;
        }
    }
}
```

## 二、二分查找

### 2.1 整数二分模板

**模板1：查找左边界**（满足条件的最小位置）

```cpp
int bsearch_left(int l, int r) {
    while (l < r) {
        int mid = (l + r) >> 1;
        if (check(mid)) r = mid;
        else l = mid + 1;
    }
    return l;
}
```

**模板2：查找右边界**（满足条件的最大位置）

```cpp
int bsearch_right(int l, int r) {
    while (l < r) {
        int mid = (l + r + 1) >> 1;
        if (check(mid)) l = mid;
        else r = mid - 1;
    }
    return l;
}
```

关键区别：当 `l = mid` 时，mid 计算要 +1（防死循环）。

### 2.2 浮点二分

```cpp
double bsearch_float(double l, double r) {
    for (int i = 0; i < 100; i++) {  // 迭代100次精度足够
        double mid = (l + r) / 2;
        if (check(mid)) r = mid;
        else l = mid;
    }
    return l;
}
```

### 2.3 二分答案

将最优化问题转化为判定问题："答案是否 ≥ x？" 对答案二分。

**经典例题**：给定 n 块木头，切成 k 段等长的，求最大长度 → 二分长度，判断能否切够 k 段。

## 三、双指针与滑动窗口

### 3.1 同向双指针

两个指针同向移动，维护一个区间。

```cpp
// 最长无重复子串
int longestUnique(string& s) {
    unordered_map<char, int> cnt;
    int res = 0;
    for (int i = 0, j = 0; i < s.size(); i++) {
        cnt[s[i]]++;
        while (cnt[s[i]] > 1) cnt[s[j++]]--;
        res = max(res, i - j + 1);
    }
    return res;
}
```

### 3.2 相向双指针

两个指针从两端向中间移动。

```cpp
// 两数之和（有序数组）
pair<int,int> twoSum(int a[], int n, int target) {
    int l = 0, r = n - 1;
    while (l < r) {
        int s = a[l] + a[r];
        if (s == target) return {l, r};
        if (s < target) l++;
        else r--;
    }
    return {-1, -1};
}
```

### 4.2 二维前缀和

```cpp
// 构建
for (int i = 1; i <= n; i++)
    for (int j = 1; j <= m; j++)
        s[i][j] = s[i-1][j] + s[i][j-1] - s[i-1][j-1] + a[i][j];

// 查询 (x1,y1) 到 (x2,y2) 的子矩阵和
int sum = s[x2][y2] - s[x1-1][y2] - s[x2][y1-1] + s[x1-1][y1-1];
```

### 4.3 差分数组


## 五、贪心算法

### 5.1 活动选择问题

n 个活动，每个有开始和结束时间，选择最多互不冲突的活动。

**贪心策略**：按结束时间排序，每次选结束最早且不冲突的。

```cpp
sort(acts.begin(), acts.end(), [](auto& a, auto& b) {
    return a.end < b.end;
});
int cnt = 0, last = -1;
for (auto& act : acts) {
    if (act.start >= last) {
        cnt++;
        last = act.end;
    }
}
```

### 5.2 区间调度

- **最多不重叠区间**：按右端点排序贪心
- **最少区间覆盖**：按左端点排序贪心
- **最少点覆盖所有区间**：按右端点排序贪心

### 5.3 贪心证明——交换论证法

1. 假设存在最优解 O，贪心解 G
2. 找到 O 与 G 第一个不同的位置
3. 证明将 O 中该位置替换为 G 的选择后，解不变差
4. 逐步替换得到 O = G，故贪心最优

## 六、分治

### 6.1 归并排序求逆序对

逆序对：i < j 且 a[i] > a[j]。在归并过程中统计。

```cpp
long long merge_count(int a[], int l, int r) {
    if (l >= r) return 0;
    int mid = (l + r) >> 1;
    long long cnt = merge_count(a, l, mid) + merge_count(a, mid + 1, r);
    int k = 0, i = l, j = mid + 1;
    while (i <= mid && j <= r) {
        if (a[i] <= a[j]) tmp[k++] = a[i++];
        else {
            tmp[k++] = a[j++];
            cnt += mid - i + 1;  // 关键：左半部分 i~mid 都大于 a[j]
        }
    }
    while (i <= mid) tmp[k++] = a[i++];
    while (j <= r) tmp[k++] = a[j++];
    for (i = l, j = 0; i <= r; i++, j++) a[i] = tmp[j];
    return cnt;
}
```

### 6.2 最近点对

将点按 x 坐标排序，分治：分别求左右两半最近点对距离 d，再检查跨中线的点对（只需检查距离中线 ≤ d 的点）。时间 O(n log n)。

## 七、位运算技巧

### 7.1 常用位运算

| 操作 | 表达式 | 含义 |
|------|--------|------|
| 判断奇偶 | `x & 1` | 1 为奇，0 为偶 |
| 除以 2 | `x >> 1` | 右移一位 |
| 乘以 2 | `x << 1` | 左移一位 |
| 取最低位 1 | `x & (-x)` | lowbit(x) |
| 去掉最低位 1 | `x & (x-1)` | 常用于统计 1 的个数 |
| 判断是否 2 的幂 | `x > 0 && (x & (x-1)) == 0` | |
| 交换两数 | `a ^= b; b ^= a; a ^= b;` | 异或交换 |
| 统计 1 的个数 | `__builtin_popcount(x)` | GCC 内建函数 |

### 7.2 状态压缩

用一个整数的每一位表示一个状态（选或不选）。

```cpp
// 枚举子集
for (int s = mask; s; s = (s - 1) & mask) {
    // s 是 mask 的一个子集
}

// 枚举 n 个元素的所有子集
for (int mask = 0; mask < (1 << n); mask++) {
    for (int i = 0; i < n; i++) {
        if (mask >> i & 1) {
            // 第 i 个元素被选中
        }
    }
}
```
