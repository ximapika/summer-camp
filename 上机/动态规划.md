# 动态规划

## 一、DP 基础

### 1.1 核心思想

动态规划（Dynamic Programming, DP）是将复杂问题分解为若干重叠子问题，通过保存子问题的解来避免重复计算。DP 适用于具有以下两个性质的问题：

- **最优子结构**：问题的最优解包含子问题的最优解。即可以通过子问题的最优解推导出原问题的最优解。
- **重叠子问题**：不同的子问题会被重复计算多次。DP 用表格保存结果，每个子问题只计算一次。

### 1.2 解题步骤

1. **定义状态**：明确 dp[i] 或 dp[i][j] 代表什么
2. **推导转移方程**：如何从已知状态计算新状态
3. **确定初始条件**：最小子问题的解
4. **确定计算顺序**：保证计算某状态时，它依赖的状态已经被计算
5. **确定答案**：最终答案是哪个状态

## 二、线性 DP

### 2.1 最长递增子序列 (LIS)

给定数组，求最长的严格递增子序列的长度。子序列不要求连续。

**O(n²) 解法**：dp[i] = 以 a[i] 结尾的 LIS 长度

```cpp
int lis(vector<int>& a) {
    int n = a.size(), ans = 0;
    vector<int> dp(n, 1);
    for (int i = 0; i < n; i++) {
        for (int j = 0; j < i; j++)
            if (a[j] < a[i])
                dp[i] = max(dp[i], dp[j] + 1);
        ans = max(ans, dp[i]);
    }
    return ans;
}
```

**O(n log n) 解法**：维护一个 tails 数组，tails[i] 是长度为 i+1 的递增子序列的最小末尾元素。

```cpp
int lis_fast(vector<int>& a) {
    vector<int> tails;
    for (int x : a) {
        auto it = lower_bound(tails.begin(), tails.end(), x);
        if (it == tails.end()) tails.push_back(x);
        else *it = x;
    }
    return tails.size();
}
```

### 2.2 最长公共子序列 (LCS)

给定两个序列 A 和 B，求最长的公共子序列（不要求连续）的长度。

状态：dp[i][j] = A 的前 i 个字符和 B 的前 j 个字符的 LCS 长度

```cpp
int lcs(string& a, string& b) {
    int n = a.size(), m = b.size();
    vector<vector<int>> dp(n+1, vector<int>(m+1, 0));
    for (int i = 1; i <= n; i++)
        for (int j = 1; j <= m; j++) {
            if (a[i-1] == b[j-1])
                dp[i][j] = dp[i-1][j-1] + 1;
            else
                dp[i][j] = max(dp[i-1][j], dp[i][j-1]);
        }
    return dp[n][m];
}
```

时间 O(nm)，空间可优化为 O(m)。

### 2.3 编辑距离 (Edit Distance)

给定两个字符串，求将一个转换为另一个所需的最少操作数（插入、删除、替换）。

状态：dp[i][j] = A 的前 i 个字符变为 B 的前 j 个字符的最小编辑距离

```cpp
int editDistance(string& a, string& b) {
    int n = a.size(), m = b.size();
    vector<vector<int>> dp(n+1, vector<int>(m+1));
    for (int i = 0; i <= n; i++) dp[i][0] = i;
    for (int j = 0; j <= m; j++) dp[0][j] = j;
    for (int i = 1; i <= n; i++)
        for (int j = 1; j <= m; j++) {
            if (a[i-1] == b[j-1])
                dp[i][j] = dp[i-1][j-1];
            else
                dp[i][j] = min({dp[i-1][j], dp[i][j-1], dp[i-1][j-1]}) + 1;
                // 分别对应删除、插入、替换
        }
    return dp[n][m];
}
```

## 三、背包问题

背包问题是 DP 中最经典的一类问题：给定容量为 W 的背包和 n 件物品，每件物品有重量 w[i] 和价值 v[i]，求背包能装下的最大价值。

### 3.1 01 背包

每件物品最多选一次。

状态：dp[j] = 容量为 j 时的最大价值

```cpp
// 一维优化版（倒序遍历保证每件物品只选一次）
for (int i = 0; i < n; i++)
    for (int j = W; j >= w[i]; j--)
        dp[j] = max(dp[j], dp[j - w[i]] + v[i]);
```

**为什么倒序？** 如果正序，dp[j-w[i]] 可能已经包含了第 i 件物品的选择，导致重复选取。倒序保证使用的是上一轮（未选第 i 件）的状态。

### 3.2 完全背包

每件物品可以选无限次。

```cpp
// 一维版（正序遍历，允许重复选取）
for (int i = 0; i < n; i++)
    for (int j = w[i]; j <= W; j++)
        dp[j] = max(dp[j], dp[j - w[i]] + v[i]);
```

### 3.3 多重背包

第 i 件物品最多选 s[i] 次。

**二进制拆分优化**：将 s[i] 拆为 1, 2, 4, ..., 余数 若干组，转化为 01 背包。复杂度 O(nW log s)。

```cpp
// 二进制拆分
for (int i = 0; i < n; i++) {
    int k = 1;
    while (k <= s[i]) {
        // 将重量 k*w[i]、价值 k*v[i] 的物品做 01 背包
        for (int j = W; j >= k * w[i]; j--)
            dp[j] = max(dp[j], dp[j - k*w[i]] + k*v[i]);
        s[i] -= k;
        k *= 2;
    }
    if (s[i] > 0) {
        for (int j = W; j >= s[i]*w[i]; j--)
            dp[j] = max(dp[j], dp[j - s[i]*w[i]] + s[i]*v[i]);
    }
}
```

### 3.4 背包问题变形

- **恰好装满**：初始化 dp[0]=0，其余 dp[j]=-INF
- **方案数统计**：dp[j] += dp[j - w[i]]
- **二维费用背包**：增加一维状态，如同时有重量和体积限制

## 四、区间 DP

### 4.1 概念

区间 DP 的状态通常定义在一个区间 [i, j] 上，通过枚举分割点 k 将区间分为 [i,k] 和 [k+1,j] 两部分来转移。遍历顺序：按区间长度从小到大。

### 4.2 石子合并

n 堆石子排成一排，每次合并相邻两堆，代价为两堆石子数之和。求最小总代价。

状态：dp[i][j] = 合并第 i 堆到第 j 堆的最小代价

```cpp
// 初始化
memset(dp, 0x3f, sizeof dp);
for (int i = 1; i <= n; i++) dp[i][i] = 0;

// 按区间长度递增
for (int len = 2; len <= n; len++)
    for (int i = 1; i + len - 1 <= n; i++) {
        int j = i + len - 1;
        for (int k = i; k < j; k++)
            dp[i][j] = min(dp[i][j], dp[i][k] + dp[k+1][j] + sum[i][j]);
            // sum[i][j] 是第 i 到 j 堆石子总数，用前缀和 O(1) 求
    }
```

时间 O(n³)。

### 4.3 矩阵链乘法

n 个矩阵相乘，不同的加括号方式导致不同的乘法次数。求最小乘法次数。与石子合并思路完全一致。

## 五、树形 DP

### 5.1 概念

在树结构上进行 DP。通常以某个节点为根，状态定义在以该节点为根的子树上，通过子节点的状态计算父节点的状态。

### 5.2 树的最大独立集

选择树中的一些节点，使得选中的节点之间没有边相连，且选中的节点权值和最大。

状态：
- dp[u][0] = 不选 u 时，u 子树的最大权值和
- dp[u][1] = 选 u 时，u 子树的最大权值和

```cpp
void dfs(int u, int fa) {
    dp[u][1] = w[u];  // 选 u，加上 u 的权值
    dp[u][0] = 0;
    for (int v : adj[u]) {
        if (v == fa) continue;
        dfs(v, u);
        dp[u][0] += max(dp[v][0], dp[v][1]);  // 不选 u，子节点可选可不选
        dp[u][1] += dp[v][0];                  // 选 u，子节点不能选
    }
}
```

### 5.3 树的直径

树中最远两点间的距离。

**方法1**：两次 BFS/DFS。从任意点出发找最远点 u，再从 u 出发找最远点 v，u-v 的距离即为直径。

**方法2**：树形 DP。dp[u] = 从 u 向下延伸的最长路径。

```cpp
int ans = 0;
void dfs(int u, int fa) {
    dp[u] = 0;
    for (int v : adj[u]) {
        if (v == fa) continue;
        dfs(v, u);
        ans = max(ans, dp[u] + dp[v] + w[u][v]);  // 经过 u 的最长路径
        dp[u] = max(dp[u], dp[v] + w[u][v]);
    }
}
```

### 5.4 换根 DP

也叫"二次扫描法"。第一次 DFS 以某个节点为根计算答案；第二次 DFS 将根转移到每个节点，利用父节点的答案 O(1) 计算当前节点的答案。适合需要求以每个节点为根时的某个值的问题。

## 六、状压 DP

### 6.1 概念

当问题的状态可以用一个集合表示时（如"哪些城市已经访问过"），可以用一个整数的二进制位来压缩表示这个集合。每一位 0/1 代表对应元素是否在集合中。适用于 n ≤ 20 的情况。

### 6.2 旅行商问题 (TSP)

n 个城市，从城市 0 出发经过所有城市恰好一次再回到起点，求最短路径。

状态：dp[S][i] = 已经访问了集合 S 中的城市，当前在城市 i 的最短路径

```cpp
int n;
int dist[20][20];
int dp[1 << 20][20];

int tsp() {
    memset(dp, 0x3f, sizeof dp);
    dp[1][0] = 0;  // 从城市 0 出发
    for (int S = 1; S < (1 << n); S++)
        for (int i = 0; i < n; i++) {
            if (!(S >> i & 1)) continue;  // i 不在 S 中
            if (dp[S][i] >= 0x3f3f3f3f) continue;
            for (int j = 0; j < n; j++) {
                if (S >> j & 1) continue;  // j 已在 S 中
                int NS = S | (1 << j);
                dp[NS][j] = min(dp[NS][j], dp[S][i] + dist[i][j]);
            }
        }
    int ans = INT_MAX;
    int full = (1 << n) - 1;
    for (int i = 0; i < n; i++)
        ans = min(ans, dp[full][i] + dist[i][0]);
    return ans;
}
```

时间 O(2ⁿ × n²)，空间 O(2ⁿ × n)。

## 七、数位 DP

### 7.1 概念

数位 DP 用于统计满足某条件的数在区间 [L, R] 中有多少个。核心思想是逐位确定数字，用"是否仍贴着上界"（tight/limit）来控制每一位的取值范围。

通常利用前缀和思想：ans(L, R) = ans(0, R) - ans(0, L-1)。

### 7.2 模板

```cpp
int a[20];  // 数字的每一位
int dp[20][...];  // 记忆化数组

int dfs(int pos, int state, bool limit, bool lead) {
    if (pos == -1) return /* 终止条件 */;
    if (!limit && !lead && dp[pos][state] != -1) return dp[pos][state];

    int up = limit ? a[pos] : 9;
    int res = 0;
    for (int d = 0; d <= up; d++) {
        // 处理前导零 lead
        res += dfs(pos - 1,
                   /* 新 state */,
                   limit && (d == up),
                   lead && (d == 0));
    }
    if (!limit && !lead) dp[pos][state] = res;
    return res;
}

int solve(int n) {
    int len = 0;
    while (n) { a[len++] = n % 10; n /= 10; }
    memset(dp, -1, sizeof dp);
    return dfs(len - 1, 0, true, true);
}
```

**经典例题**：统计 [1, N] 中不包含数字 4 的数的个数。

## 八、DP 优化

### 8.1 滚动数组

当 dp[i] 只依赖 dp[i-1] 时，可以只保留两行（或一行），空间从 O(n × m) 降为 O(m)。

```cpp
// 原始：dp[i][j] = dp[i-1][j-1] + dp[i-1][j]
// 优化：只用一行，倒序遍历
for (int i = 1; i <= n; i++)
    for (int j = W; j >= w[i]; j--)  // 倒序！
        dp[j] = max(dp[j], dp[j - w[i]] + v[i]);
```

### 8.2 单调队列优化

当转移方程形如 dp[i] = min/max(dp[j] + ...) 且 j 的范围是一个滑动窗口时，可以用单调队列将转移从 O(k) 优化到 O(1)。

**经典应用**：多重背包的单调队列优化、最大子序和的限制长度版本。

### 8.3 斜率优化

当转移方程可以化为 dp[i] = min(dp[j] + f(i,j)) 的形式，且 f(i,j) 可以分离变量时，用斜率优化将 O(n²) 降为 O(n)。

核心思想：将转移方程改写为 y = kx + b 的形式，维护下凸壳（最小值）或上凸壳（最大值），用单调队列维护。

属于进阶技巧，了解思想即可。
