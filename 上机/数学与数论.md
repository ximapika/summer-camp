## 1. 快速幂

### 1.1 整数快速幂

**核心思想**：利用指数的二进制分解，将 a^n 的计算从 O(n) 优化到 O(log n)。例如 a^13 = a^(1101_2) = a^8 * a^4 * a^1。每一步对底数平方，检查指数最低位决定是否乘入结果。

**时间复杂度**：O(log n)

```cpp
long long qpow(long long a, long long n, long long mod) {
    long long res = 1;
    a %= mod;
    while (n > 0) {
        if (n & 1) res = res * a % mod;
        a = a * a % mod;
        n >>= 1;
    }
    return res;
}
```

### 1.2 矩阵快速幂

**核心思想**：将快速幂推广到矩阵乘法。利用矩阵乘法的结合律，O(k^3 log n) 计算矩阵的 n 次幂（k 为矩阵维度）。常用于加速线性递推。

**时间复杂度**：O(k^3 log n)，k 为矩阵大小

```cpp
typedef vector<vector<long long>> Matrix;
const long long MOD = 1e9 + 7;

Matrix multiply(const Matrix& A, const Matrix& B) {
    int n = A.size(), m = B[0].size(), p = B.size();
    Matrix C(n, vector<long long>(m, 0));
    for (int i = 0; i < n; i++)
        for (int k = 0; k < p; k++)
            for (int j = 0; j < m; j++)
                C[i][j] = (C[i][j] + A[i][k] * B[k][j]) % MOD;
    return C;
}

Matrix matpow(Matrix A, long long n) {
    int sz = A.size();
    Matrix res(sz, vector<long long>(sz, 0));
    for (int i = 0; i < sz; i++) res[i][i] = 1; // 单位矩阵
    while (n > 0) {
        if (n & 1) res = multiply(res, A);
        A = multiply(A, A);
        n >>= 1;
    }
    return res;
}
```

**经典例题**：斐波那契数列加速
- F(n) = F(n-1) + F(n-2)，构造转移矩阵 `[[1,1],[1,0]]`，则 `[[F(n+1)],[F(n)]] = M^n * [[F(1)],[F(0)]]`。
- 可以 O(log n) 求第 n 项 Fibonacci 数。

```cpp
// 求第 n 个 Fibonacci 数 mod p
long long fibonacci(long long n) {
    if (n <= 1) return n;
    Matrix M = {{1, 1}, {1, 0}};
    Matrix R = matpow(M, n - 1);
    return R[0][0]; // F(n)
}
```

**扩展应用**：
- 任意 k 阶线性递推：f(n) = c1*f(n-1) + c2*f(n-2) + ... + ck*f(n-k)
- 图上走 k 步的路径计数：邻接矩阵的 k 次幂

## 2. GCD 与 LCM

### 2.1 欧几里得算法（辗转相除法）

**核心思想**：gcd(a, b) = gcd(b, a % b)，递归直到 b = 0。

**时间复杂度**：O(log(min(a, b)))

```cpp
int gcd(int a, int b) {
    return b == 0 ? a : gcd(b, a % b);
}

int lcm(int a, int b) {
    return a / gcd(a, b) * b; // 先除后乘防溢出
}
```

### 2.2 扩展欧几里得算法

**核心思想**：在求 gcd(a, b) 的同时，找到整数 x, y 使得 `ax + by = gcd(a, b)`。递归回溯时更新 x, y。

**时间复杂度**：O(log(min(a, b)))

```cpp
int exgcd(int a, int b, int& x, int& y) {
    if (b == 0) {
        x = 1; y = 0;
        return a;
    }
    int d = exgcd(b, a % b, y, x);
    y -= (a / b) * x;
    return d;
}
```

**应用**：
- 求 `ax + by = c` 的解：当且仅当 `gcd(a,b) | c` 时有解。先求 `ax + by = gcd(a,b)` 的解，再乘以 `c / gcd(a,b)`。
- 求模逆元：`ax ≡ 1 (mod m)` 等价于 `ax + my = 1`，当 gcd(a, m) = 1 时有解。

### 2.3 裴蜀定理（Bezout's Identity）

**定理**：对任意整数 a, b，存在整数 x, y 使得 `ax + by = gcd(a, b)`。推论：`ax + by = c` 有整数解的充要条件是 `gcd(a,b) | c`。

**经典例题**：水壶问题（LeetCode 365）
- 两个水壶容量分别为 a 和 b，能否量出 c 升水？等价于判断 `gcd(a,b) | c`。

## 3. 素数

### 3.1 素数判定

**朴素方法**：试除法，检查 2 到 sqrt(n) 的所有因子。

**时间复杂度**：O(sqrt(n))

```cpp
bool isPrime(int n) {
    if (n < 2) return false;
    for (int i = 2; (long long)i * i <= n; i++)
        if (n % i == 0) return false;
    return true;
}
```

### 3.2 埃氏筛（Sieve of Eratosthenes）

**核心思想**：从小到大遍历，将每个素数的所有倍数标记为合数。

**时间复杂度**：O(n log log n)

```cpp
const int MAXN = 10000005;
bool notPrime[MAXN];
vector<int> primes;

void sieve(int n) {
    notPrime[0] = notPrime[1] = true;
    for (int i = 2; i <= n; i++) {
        if (!notPrime[i]) {
            primes.push_back(i);
            for (long long j = (long long)i * i; j <= n; j += i)
                notPrime[j] = true;
        }
    }
}
```

### 3.3 线性筛（欧拉筛）

**核心思想**：保证每个合数只被其**最小质因子**筛掉一次，达到严格 O(n)。关键：当 `i % primes[j] == 0` 时 break，因为之后的合数 `i * primes[j+1]` 的最小质因子是 `primes[j]` 而非 `primes[j+1]`。

**时间复杂度**：O(n)

```cpp
bool notPrime[MAXN];
int primes[MAXN], cnt = 0;

void linearSieve(int n) {
    notPrime[0] = notPrime[1] = true;
    for (int i = 2; i <= n; i++) {
        if (!notPrime[i]) primes[cnt++] = i;
        for (int j = 0; j < cnt && (long long)i * primes[j] <= n; j++) {
            notPrime[i * primes[j]] = true;
            if (i % primes[j] == 0) break; // 关键！
        }
    }
}
```

**经典例题**：求 1 到 n 之间的素数个数，或求所有素数。

**线性筛的扩展**：可以在线性筛的过程中同时求出欧拉函数 phi(n)、莫比乌斯函数 mu(n) 等积性函数。

```cpp
int phi[MAXN];

void sieveWithPhi(int n) {
    phi[1] = 1;
    for (int i = 2; i <= n; i++) {
        if (!notPrime[i]) {
            primes[cnt++] = i;
            phi[i] = i - 1; // 素数的欧拉函数
        }
        for (int j = 0; j < cnt && (long long)i * primes[j] <= n; j++) {
            notPrime[i * primes[j]] = true;
            if (i % primes[j] == 0) {
                phi[i * primes[j]] = phi[i] * primes[j];
                break;
            }
            phi[i * primes[j]] = phi[i] * (primes[j] - 1);
        }
    }
}
```

## 4. 模运算

### 4.1 模逆元

**概念**：若 `a * x ≡ 1 (mod m)`，则 x 是 a 在模 m 下的逆元，记为 a^(-1)。要求 gcd(a, m) = 1。

**方法一：费马小定理**（m 为素数时）

若 m 为素数，则 `a^(m-1) ≡ 1 (mod m)`，所以 `a^(-1) ≡ a^(m-2) (mod m)`。

```cpp
long long inv(long long a, long long m) {
    return qpow(a, m - 2, m); // 需要快速幂
}
```

**方法二：扩展欧几里得**（通用方法）

```cpp
long long inv(long long a, long long m) {
    long long x, y;
    int d = exgcd(a, m, x, y);
    // d 应为 1（a 与 m 互素）
    return (x % m + m) % m;
}
```

**方法三：线性求逆元**（求 1 到 n 所有数模素数 p 的逆元）

```cpp
long long inv[MAXN];

void preInv(int n, long long p) {
    inv[1] = 1;
    for (int i = 2; i <= n; i++)
        inv[i] = (p - p / i) * inv[p % i] % p;
}
```

### 4.2 中国剩余定理（CRT, Chinese Remainder Theorem）

**定理**：给定两两互素的模数 m1, m2, ..., mk，以及方程组：
```
x ≡ a1 (mod m1)
x ≡ a2 (mod m2)
...
x ≡ ak (mod mk)
```
则在模 M = m1 * m2 * ... * mk 下有唯一解。

**时间复杂度**：O(k log M)

```cpp
// 经典 CRT：模数两两互素
long long CRT(vector<long long>& a, vector<long long>& m) {
    int k = a.size();
    long long M = 1;
    for (int i = 0; i < k; i++) M *= m[i];
    long long res = 0;
    for (int i = 0; i < k; i++) {
        long long Mi = M / m[i];
        long long x, y;
        exgcd(Mi, m[i], x, y);
        res = (res + a[i] % M * (Mi % M) % M * (x % M + M) % M) % M;
    }
    return (res + M) % M;
}
```

**扩展 CRT**（模数不互素时）：逐步合并方程，使用 exgcd。

```cpp
// 扩展 CRT：模数不要求互素
// 将 x ≡ a1 (mod m1) 和 x ≡ a2 (mod m2) 合并为 x ≡ a3 (mod m3)
// 返回 {a3, m3}，无解返回 {-1, -1}
pair<long long, long long> merge(long long a1, long long m1, long long a2, long long m2) {
    long long x, y;
    long long d = exgcd(m1, m2, x, y);
    if ((a2 - a1) % d != 0) return {-1, -1}; // 无解
    long long lcm = m1 / d * m2;
    long long t = (a2 - a1) / d % (m2 / d) * x % (m2 / d);
    long long a3 = ((a1 + m1 * t) % lcm + lcm) % lcm;
    return {a3, lcm};
}

long long exCRT(vector<long long>& a, vector<long long>& m) {
    long long curA = a[0], curM = m[0];
    for (int i = 1; i < (int)a.size(); i++) {
        auto [na, nm] = merge(curA, curM, a[i], m[i]);
        if (nm == -1) return -1; // 无解
        curA = na; curM = nm;
    }
    return curA;
}
```

**经典例题**：韩信点兵 / 孙子定理
- 一个数除以 3 余 2，除以 5 余 3，除以 7 余 2，求这个数。

## 5. 组合数学

### 5.1 排列组合公式

- **排列数**：A(n, k) = n! / (n-k)!
- **组合数**：C(n, k) = n! / (k! * (n-k)!)
- 递推：C(n, k) = C(n-1, k-1) + C(n-1, k)

```cpp
// 预处理阶乘和逆元，O(1) 求 C(n, k) mod p（p 为素数）
const int MAXN = 200005;
const long long MOD = 1e9 + 7;
long long fac[MAXN], inv_fac[MAXN];

void initComb(int n) {
    fac[0] = 1;
    for (int i = 1; i <= n; i++) fac[i] = fac[i-1] * i % MOD;
    inv_fac[n] = qpow(fac[n], MOD - 2, MOD);
    for (int i = n - 1; i >= 0; i--) inv_fac[i] = inv_fac[i+1] * (i+1) % MOD;
}

long long C(int n, int k) {
    if (k < 0 || k > n) return 0;
    return fac[n] % MOD * inv_fac[k] % MOD * inv_fac[n-k] % MOD;
}
```

### 5.2 Lucas 定理

**定理**：当 p 为素数且 n, k 可能很大时：
```
C(n, k) mod p = C(n mod p, k mod p) * C(n/p, k/p) mod p
```

**适用条件**：p 为素数且 p 不太大（能预处理 0 到 p-1 的阶乘）。

**时间复杂度**：O(p + log_p(n))

```cpp
long long lucas(long long n, long long k, long long p) {
    if (k == 0) return 1;
    return C(n % p, k % p) * lucas(n / p, k / p, p) % p;
}
```

**注意**：这里的 C 函数需要能处理 n, k < p 的情况（使用上面预处理的版本）。

### 5.3 卡特兰数（Catalan Number）

**公式**：
- `Cat(n) = C(2n, n) / (n+1)`
- `Cat(n) = C(2n, n) - C(2n, n-1)`
- 递推：`Cat(0) = 1, Cat(n) = sum(Cat(i) * Cat(n-1-i), i=0..n-1)`

**前几项**：1, 1, 2, 5, 14, 42, 132, 429, ...

**经典应用**：
- n 对合法括号序列的数量
- n 个节点的不同二叉搜索树数量
- n+1 个矩阵链乘的不同加括号方式
- 从 (0,0) 到 (n,n) 不超过对角线的格路数
- 凸多边形三角剖分数

```cpp
long long catalan(int n) {
    return C(2 * n, n) % MOD * qpow(n + 1, MOD - 2, MOD) % MOD;
}
```

### 5.4 容斥原理（Inclusion-Exclusion Principle）

**公式**：
```
|A1 ∪ A2 ∪ ... ∪ An| = Σ|Ai| - Σ|Ai ∩ Aj| + Σ|Ai ∩ Aj ∩ Ak| - ... + (-1)^(n+1)|A1 ∩ ... ∩ An|
```

**时间复杂度**：O(2^n)，n 为集合个数

```cpp
// 求 1 到 m 中至少能被集合 a[] 中某个数整除的数的个数
long long inclusionExclusion(vector<int>& a, long long m) {
    int n = a.size();
    long long res = 0;
    for (int mask = 1; mask < (1 << n); mask++) {
        long long lcm_val = 1;
        int bits = 0;
        for (int i = 0; i < n; i++) {
            if (mask & (1 << i)) {
                bits++;
                lcm_val = lcm(lcm_val, (long long)a[i]);
                if (lcm_val > m) break; // 防溢出
            }
        }
        if (bits % 2 == 1)
            res += m / lcm_val;
        else
            res -= m / lcm_val;
    }
    return res;
}
```

**经典例题**：
- 错位排列（错排数）：D(n) = n! * Σ (-1)^k / k!, k=0..n
  - 递推：D(n) = (n-1) * (D(n-1) + D(n-2))
- 求 1 到 n 中与 m 互素的数的个数：对 m 的质因子集合做容斥

## 6. 博弈论

### 6.1 Nim 游戏

**规则**：有若干堆石子，两个玩家轮流取，每次从某一堆取任意正数个。取走最后一个石子的玩家获胜。

**Nim 定理**：先手必胜当且仅当所有堆石子数的**异或和不为 0**。

```cpp
bool nimWin(vector<int>& piles) {
    int xorSum = 0;
    for (int x : piles) xorSum ^= x;
    return xorSum != 0; // true 表示先手必胜
}
```

**证明思路**：
- 异或和为 0 的状态是 P-position（后手必胜），因为任何操作后异或和变为非 0。
- 异或和非 0 的状态是 N-position（先手必胜），因为总存在一种操作使异或和变为 0。

### 6.2 SG 函数（Sprague-Grundy 定理）

**核心思想**：
- 每个游戏状态有一个 SG 值：`SG(x) = mex({SG(y) | y 是 x 的后继状态})`
- mex（最小排除值）：集合中未出现的最小非负整数。
- **终止状态的 SG 值为 0**（无后继）。
- **多个独立子游戏的 SG 值等于各子游戏 SG 值的异或**。
- SG 值为 0 表示当前状态后手必胜（P-position），非 0 表示先手必胜（N-position）。

```cpp
// 通用 SG 函数计算（记忆化搜索）
int sg[MAXN];
bool vis_sg[MAXN];

int getSG(int x) {
    if (vis_sg[x]) return sg[x];
    vis_sg[x] = true;
    set<int> reachable;
    // 枚举所有后继状态
    for (int next : getSuccessors(x)) {
        reachable.insert(getSG(next));
    }
    // 求 mex
    int mex = 0;
    while (reachable.count(mex)) mex++;
    return sg[x] = mex;
}
```

**经典例题**：取石子变形
- 每次可以取 1, 3, 4 个石子。SG(0)=0, SG(1)=1, SG(2)=0, SG(3)=2, SG(4)=3, SG(5)=1, ...
- 多堆时对每堆的 SG 值异或，非 0 则先手胜。

**经典博弈类型总结**：

| 博弈类型 | 规则 | 先手必胜条件 |
|---------|------|------------|
| Nim | 任取 | 异或和 != 0 |
| Bash | 每次最多取 m 个 | n % (m+1) != 0 |
| Wythoff | 两堆，可取一堆任意或两堆相同 | (a, b) 不是冷局 |
| 阶梯 Nim | 奇数阶梯上的异或 | 奇数层异或和 != 0 |

## 7. 高精度运算

### 7.1 大整数表示与加法

**核心思想**：用数组（vector）存储每一位数字，低位在前。模拟手算过程逐位运算，处理进位/借位。

```cpp
typedef vector<int> BigInt;

// 字符串转大整数（低位在前）
BigInt fromString(const string& s) {
    BigInt a;
    for (int i = s.size() - 1; i >= 0; i--)
        a.push_back(s[i] - '0');
    return a;
}

// 大整数转字符串
string toString(const BigInt& a) {
    string s;
    for (int i = a.size() - 1; i >= 0; i--)
        s += (char)('0' + a[i]);
    if (s.empty()) s = "0";
    return s;
}

// 高精度加法
BigInt add(const BigInt& a, const BigInt& b) {
    BigInt c;
    int carry = 0;
    for (int i = 0; i < (int)max(a.size(), b.size()) || carry; i++) {
        int sum = carry;
        if (i < (int)a.size()) sum += a[i];
        if (i < (int)b.size()) sum += b[i];
        c.push_back(sum % 10);
        carry = sum / 10;
    }
    return c;
}
```

### 7.2 高精度减法

```cpp
// 比较大小：a >= b 返回 true
bool geq(const BigInt& a, const BigInt& b) {
    if (a.size() != b.size()) return a.size() > b.size();
    for (int i = a.size() - 1; i >= 0; i--)
        if (a[i] != b[i]) return a[i] > b[i];
    return true;
}

// 高精度减法（要求 a >= b）
BigInt sub(const BigInt& a, const BigInt& b) {
    BigInt c;
    int borrow = 0;
    for (int i = 0; i < (int)a.size(); i++) {
        int diff = a[i] - borrow;
        if (i < (int)b.size()) diff -= b[i];
        if (diff < 0) { diff += 10; borrow = 1; }
        else borrow = 0;
        c.push_back(diff);
    }
    while (c.size() > 1 && c.back() == 0) c.pop_back(); // 去前导零
    return c;
}
```

### 7.3 高精度乘法

```cpp
// 高精度 * 低精度
BigInt mulSmall(const BigInt& a, int b) {
    BigInt c;
    int carry = 0;
    for (int i = 0; i < (int)a.size() || carry; i++) {
        long long cur = carry;
        if (i < (int)a.size()) cur += (long long)a[i] * b;
        c.push_back(cur % 10);
        carry = cur / 10;
    }
    while (c.size() > 1 && c.back() == 0) c.pop_back();
    return c;
}

// 高精度 * 高精度
BigInt mul(const BigInt& a, const BigInt& b) {
    BigInt c(a.size() + b.size(), 0);
    for (int i = 0; i < (int)a.size(); i++) {
        int carry = 0;
        for (int j = 0; j < (int)b.size() || carry; j++) {
            long long cur = c[i + j] + carry;
            if (j < (int)b.size()) cur += (long long)a[i] * b[j];
            c[i + j] = cur % 10;
            carry = cur / 10;
        }
    }
    while (c.size() > 1 && c.back() == 0) c.pop_back();
    return c;
}
```

### 7.4 高精度除法

```cpp
// 高精度 / 低精度，返回 {商, 余数}
pair<BigInt, int> divSmall(const BigInt& a, int b) {
    BigInt c;
    int remainder = 0;
    for (int i = a.size() - 1; i >= 0; i--) {
        remainder = remainder * 10 + a[i];
        c.push_back(remainder / b);
        remainder %= b;
    }
    reverse(c.begin(), c.end());
    while (c.size() > 1 && c.back() == 0) c.pop_back();
    return {c, remainder};
}
```

**经典例题**：
- **大数阶乘**：求 n! 的精确值（n 可达数千），用高精度乘法逐步乘。
- **大数幂**：配合快速幂和高精度乘法。
- **A + B Problem**（洛谷 P1601）：两个不超过 200 位的非负整数相加。
