## 1. 图的存储

### 1.1 邻接矩阵

**核心思想**：用二维数组 `g[i][j]` 存储节点 i 到节点 j 的边权。适合稠密图。

**时间复杂度**：查询 O(1)，遍历邻居 O(V)，空间 O(V^2)。

```cpp
const int MAXN = 1005;
int g[MAXN][MAXN]; // g[i][j] = w 表示边权，0 或 INF 表示无边

void addEdge(int u, int v, int w) {
    g[u][v] = w;
    // g[v][u] = w; // 无向图
}
```

### 1.2 邻接表（vector 实现）

**核心思想**：每个节点用一个 vector 存储所有出边。适合稀疏图，最常用。

**时间复杂度**：空间 O(V + E)，遍历邻居 O(度数)。

```cpp
struct Edge {
    int to, w;
};
vector<Edge> adj[MAXN];

void addEdge(int u, int v, int w) {
    adj[u].push_back({v, w});
    // adj[v].push_back({u, w}); // 无向图
}
```

### 1.3 链式前向星

**核心思想**：用数组模拟链表存图，常数极小，适合卡常或需要边编号的场景（如网络流）。

**时间复杂度**：同邻接表。

```cpp
const int MAXN = 100005, MAXM = 200005;
int head[MAXN], cnt = 0;
struct Edge {
    int to, w, nxt;
} e[MAXM];

void addEdge(int u, int v, int w) {
    e[++cnt] = {v, w, head[u]};
    head[u] = cnt;
}

// 遍历 u 的所有出边
// for (int i = head[u]; i; i = e[i].nxt) { int v = e[i].to, w = e[i].w; }
```

## 2. 最短路径

### 2.1 Dijkstra（堆优化）

**核心思想**：贪心策略，每次取出距离源点最近的未确定节点进行松弛。要求**无负权边**。堆优化用优先队列维护最小距离。

**时间复杂度**：O((V + E) log V)

```cpp
#include <bits/stdc++.h>
using namespace std;
typedef pair<int,int> pii;
const int INF = 0x3f3f3f3f;

vector<pii> adj[MAXN]; // {to, weight}
int dist[MAXN];
bool vis[MAXN];

void dijkstra(int s, int n) {
    memset(dist, 0x3f, sizeof(dist));
    memset(vis, false, sizeof(vis));
    dist[s] = 0;
    priority_queue<pii, vector<pii>, greater<pii>> pq;
    pq.push({0, s});
    while (!pq.empty()) {
        auto [d, u] = pq.top(); pq.pop();
        if (vis[u]) continue;
        vis[u] = true;
        for (auto [v, w] : adj[u]) {
            if (dist[u] + w < dist[v]) {
                dist[v] = dist[u] + w;
                pq.push({dist[v], v});
            }
        }
    }
}
```

**经典例题**：单源最短路径（洛谷 P4779）
- 给定有向图，求从源点 s 到所有点的最短距离。直接套用 Dijkstra 堆优化模板。

### 2.2 Bellman-Ford

**核心思想**：对所有边进行 V-1 轮松弛。可处理**负权边**，还能检测**负环**（第 V 轮仍能松弛则存在负环）。

**时间复杂度**：O(VE)

```cpp
struct Edge { int u, v, w; };
vector<Edge> edges;
int dist[MAXN];

bool bellmanFord(int s, int n) {
    memset(dist, 0x3f, sizeof(dist));
    dist[s] = 0;
    for (int i = 1; i <= n - 1; i++) {
        for (auto& e : edges) {
            if (dist[e.u] != INF && dist[e.u] + e.w < dist[e.v]) {
                dist[e.v] = dist[e.u] + e.w;
            }
        }
    }
    // 检测负环
    for (auto& e : edges) {
        if (dist[e.u] != INF && dist[e.u] + e.w < dist[e.v])
            return true; // 存在负环
    }
    return false;
}
```

**经典例题**：判断负环
- 给定有向图，判断是否存在从源点可达的负权环。第 V 轮松弛仍有更新则负环存在。

### 2.3 SPFA（Shortest Path Faster Algorithm）

**核心思想**：Bellman-Ford 的队列优化版本。用队列维护"距离被更新的节点"，只松弛这些节点的出边。平均表现好但最坏退化为 O(VE)。

**时间复杂度**：平均 O(kE)，k 为小常数；最坏 O(VE)

```cpp
int dist[MAXN], cnt[MAXN]; // cnt[i] 记录 i 入队次数，用于判负环
bool inq[MAXN];

bool spfa(int s, int n) {
    memset(dist, 0x3f, sizeof(dist));
    memset(inq, false, sizeof(inq));
    memset(cnt, 0, sizeof(cnt));
    dist[s] = 0;
    queue<int> q;
    q.push(s); inq[s] = true; cnt[s] = 1;
    while (!q.empty()) {
        int u = q.front(); q.pop();
        inq[u] = false;
        for (auto [v, w] : adj[u]) {
            if (dist[u] + w < dist[v]) {
                dist[v] = dist[u] + w;
                if (!inq[v]) {
                    q.push(v);
                    inq[v] = true;
                    if (++cnt[v] > n) return true; // 负环
                }
            }
        }
    }
    return false;
}
```

**经典例题**：判断负环（洛谷 P3385）
- 如果某节点入队超过 n 次，则存在负环。

### 2.4 Floyd

**核心思想**：动态规划，枚举中间节点 k，考虑经过 k 的路径是否更短。求**全源最短路**。可处理负权边（不能有负环）。

**时间复杂度**：O(V^3)，空间 O(V^2)

```cpp
int d[MAXN][MAXN]; // 初始化为 INF，d[i][i] = 0

void floyd(int n) {
    for (int k = 1; k <= n; k++)
        for (int i = 1; i <= n; i++)
            for (int j = 1; j <= n; j++)
                d[i][j] = min(d[i][j], d[i][k] + d[k][j]);
}
```

**经典例题**：多源最短路 / 传递闭包
- 给定城市之间的距离，求任意两城市间的最短路。直接套模板。
- 传递闭包：将 `min` 改为 `||`（逻辑或），判断可达性。

## 3. 最小生成树

### 3.1 Prim 算法

**核心思想**：从一个点出发，每次选取连接已选集合和未选集合的**最小权边**，加入生成树。类似 Dijkstra 的贪心思路。

**时间复杂度**：朴素 O(V^2)，堆优化 O((V + E) log V)

```cpp
// 堆优化 Prim
typedef pair<int,int> pii;

int prim(int n) {
    vector<int> dist(n + 1, INF);
    vector<bool> vis(n + 1, false);
    priority_queue<pii, vector<pii>, greater<pii>> pq;
    dist[1] = 0;
    pq.push({0, 1});
    int res = 0, cnt = 0;
    while (!pq.empty()) {
        auto [d, u] = pq.top(); pq.pop();
        if (vis[u]) continue;
        vis[u] = true;
        res += d; cnt++;
        for (auto [v, w] : adj[u]) {
            if (!vis[v] && w < dist[v]) {
                dist[v] = w;
                pq.push({w, v});
            }
        }
    }
    return cnt == n ? res : -1; // -1 表示图不连通
}
```

### 3.2 Kruskal 算法（并查集）

**核心思想**：将所有边按权值排序，依次选边，若边的两端不在同一连通分量则加入（用并查集判断），直到选够 V-1 条边。

**时间复杂度**：O(E log E)

```cpp
struct Edge {
    int u, v, w;
    bool operator<(const Edge& o) const { return w < o.w; }
};

int fa[MAXN];
int find(int x) { return fa[x] == x ? x : fa[x] = find(fa[x]); }

int kruskal(int n, vector<Edge>& edges) {
    sort(edges.begin(), edges.end());
    for (int i = 1; i <= n; i++) fa[i] = i;
    int res = 0, cnt = 0;
    for (auto& e : edges) {
        int fu = find(e.u), fv = find(e.v);
        if (fu != fv) {
            fa[fu] = fv;
            res += e.w;
            if (++cnt == n - 1) break;
        }
    }
    return cnt == n - 1 ? res : -1;
}
```

**经典例题**：最小生成树（洛谷 P3366）
- 给定无向连通图，求最小生成树的总边权。Kruskal 更适合稀疏图，Prim 更适合稠密图。

## 4. 拓扑排序

### 4.1 Kahn 算法（BFS）

**核心思想**：维护入度为 0 的节点队列，每次取出一个节点加入排序结果，并将其出边终点的入度减 1。若减为 0 则入队。最终结果节点数不等于 V 则有环。

**时间复杂度**：O(V + E)

```cpp
vector<int> adj[MAXN];
int indeg[MAXN];

vector<int> topoSort(int n) {
    queue<int> q;
    for (int i = 1; i <= n; i++)
        if (indeg[i] == 0) q.push(i);
    vector<int> order;
    while (!q.empty()) {
        int u = q.front(); q.pop();
        order.push_back(u);
        for (int v : adj[u]) {
            if (--indeg[v] == 0) q.push(v);
        }
    }
    return order; // order.size() == n 则无环
}
```

### 4.2 DFS 实现

**核心思想**：对每个未访问节点做 DFS，回溯时将节点压入栈。最终栈的逆序就是拓扑序。可通过检测回边判断环。

```cpp
int color[MAXN]; // 0=白, 1=灰(栈中), 2=黑(完成)
vector<int> order;
bool hasCycle;

void dfs(int u) {
    color[u] = 1;
    for (int v : adj[u]) {
        if (color[v] == 1) { hasCycle = true; return; }
        if (color[v] == 0) dfs(v);
    }
    color[u] = 2;
    order.push_back(u);
}

vector<int> topoSortDFS(int n) {
    hasCycle = false;
    memset(color, 0, sizeof(color));
    order.clear();
    for (int i = 1; i <= n; i++)
        if (color[i] == 0) dfs(i);
    reverse(order.begin(), order.end());
    return order;
}
```

**经典例题**：课程表（LeetCode 207 / 210）
- 给定 n 门课和先修关系（有向图），判断能否完成所有课程（判环），或输出一种合法学习顺序（拓扑排序）。

## 5. 强连通分量（SCC）

### 5.1 Tarjan 算法

**核心思想**：利用 DFS 时间戳和栈。每个节点维护 `dfn`（发现时间）和 `low`（通过子树内的边能追溯到的最早时间戳）。当 `dfn[u] == low[u]` 时，u 是一个 SCC 的根，弹栈得到该 SCC 的所有节点。

**时间复杂度**：O(V + E)

```cpp
int dfn[MAXN], low[MAXN], timer = 0;
int stk[MAXN], top = 0;
bool inStk[MAXN];
int sccId[MAXN], sccCnt = 0;

void tarjan(int u) {
    dfn[u] = low[u] = ++timer;
    stk[++top] = u; inStk[u] = true;
    for (int v : adj[u]) {
        if (!dfn[v]) {
            tarjan(v);
            low[u] = min(low[u], low[v]);
        } else if (inStk[v]) {
            low[u] = min(low[u], dfn[v]);
        }
    }
    if (dfn[u] == low[u]) {
        sccCnt++;
        while (true) {
            int v = stk[top--];
            inStk[v] = false;
            sccId[v] = sccCnt;
            if (v == u) break;
        }
    }
}

void findSCC(int n) {
    memset(dfn, 0, sizeof(dfn));
    timer = 0; sccCnt = 0; top = 0;
    for (int i = 1; i <= n; i++)
        if (!dfn[i]) tarjan(i);
}
```

### 5.2 Kosaraju 算法

**核心思想**：两次 DFS。第一次在原图上 DFS，按完成时间逆序排列节点。第二次在反图上按此顺序 DFS，每次 DFS 访问到的节点构成一个 SCC。

**时间复杂度**：O(V + E)

```cpp
vector<int> adj[MAXN], radj[MAXN]; // 原图和反图
bool vis[MAXN];
vector<int> order;
int comp[MAXN], sccCnt;

void dfs1(int u) {
    vis[u] = true;
    for (int v : adj[u]) if (!vis[v]) dfs1(v);
    order.push_back(u);
}

void dfs2(int u, int c) {
    comp[u] = c;
    for (int v : radj[u]) if (!comp[v]) dfs2(v, c);
}

void kosaraju(int n) {
    memset(vis, false, sizeof(vis));
    order.clear();
    for (int i = 1; i <= n; i++) if (!vis[i]) dfs1(i);
    sccCnt = 0;
    memset(comp, 0, sizeof(comp));
    for (int i = (int)order.size() - 1; i >= 0; i--) {
        if (!comp[order[i]])
            dfs2(order[i], ++sccCnt);
    }
}
```

**经典例题**：缩点（洛谷 P3387）
- 给定有向图，每个节点有权值，求一条路径使得经过的不同 SCC 的权值之和最大。先 Tarjan 缩点，再在 DAG 上 DP。

## 6. 二分图

### 6.1 二分图判定（染色法）

**核心思想**：用 BFS/DFS 对图进行黑白染色。若相邻节点同色，则不是二分图。图是二分图当且仅当不含奇环。

**时间复杂度**：O(V + E)

```cpp
int color[MAXN]; // -1 未染色, 0/1 两色

bool isBipartite(int n) {
    memset(color, -1, sizeof(color));
    for (int i = 1; i <= n; i++) {
        if (color[i] != -1) continue;
        queue<int> q;
        q.push(i); color[i] = 0;
        while (!q.empty()) {
            int u = q.front(); q.pop();
            for (int v : adj[u]) {
                if (color[v] == -1) {
                    color[v] = color[u] ^ 1;
                    q.push(v);
                } else if (color[v] == color[u]) {
                    return false;
                }
            }
        }
    }
    return true;
}
```

### 6.2 最大匹配（匈牙利算法）

**核心思想**：对左部每个节点尝试找增广路。增广路是一条从未匹配节点出发，交替经过非匹配边和匹配边，到达另一个未匹配节点的路径。找到增广路则匹配数加 1。

**时间复杂度**：O(VE)

```cpp
int match_r[MAXN]; // 右部节点的匹配对象
bool vis[MAXN];

bool dfs(int u) {
    for (int v : adj[u]) {
        if (vis[v]) continue;
        vis[v] = true;
        if (match_r[v] == 0 || dfs(match_r[v])) {
            match_r[v] = u;
            return true;
        }
    }
    return false;
}

int hungarian(int n_left) {
    memset(match_r, 0, sizeof(match_r));
    int res = 0;
    for (int u = 1; u <= n_left; u++) {
        memset(vis, false, sizeof(vis));
        if (dfs(u)) res++;
    }
    return res;
}
```

**经典例题**：棋盘覆盖
- 在一个 n*m 的棋盘上放 1x2 的骨牌，某些格子不能放。将格子按 (i+j) 奇偶性分为两部分建二分图，求最大匹配即为答案。

**重要定理**：
- **最小点覆盖 = 最大匹配**（Konig 定理）
- **最大独立集 = V - 最大匹配**

## 7. 网络流

### 7.1 最大流 - Dinic 算法

**核心思想**：
1. BFS 建分层图（按距离分层）。
2. DFS 在分层图上找阻塞流（多路增广）。
3. 重复直到无法从源到汇。
4. 使用**当前弧优化**避免重复搜索无用边。

**时间复杂度**：O(V^2 * E)，二分图上为 O(E * sqrt(V))

```cpp
struct Edge {
    int to, cap, rev; // rev: 反向边在 adj[to] 中的下标
};
vector<Edge> adj[MAXN];
int level[MAXN], iter[MAXN];

void addEdge(int u, int v, int cap) {
    adj[u].push_back({v, cap, (int)adj[v].size()});
    adj[v].push_back({u, 0, (int)adj[u].size() - 1}); // 反向边容量为 0
}

bool bfs(int s, int t) {
    memset(level, -1, sizeof(level));
    queue<int> q;
    level[s] = 0; q.push(s);
    while (!q.empty()) {
        int u = q.front(); q.pop();
        for (auto& e : adj[u]) {
            if (e.cap > 0 && level[e.to] < 0) {
                level[e.to] = level[u] + 1;
                q.push(e.to);
            }
        }
    }
    return level[t] >= 0;
}

int dfs(int u, int t, int f) {
    if (u == t) return f;
    for (int& i = iter[u]; i < (int)adj[u].size(); i++) {
        Edge& e = adj[u][i];
        if (e.cap > 0 && level[e.to] == level[u] + 1) {
            int d = dfs(e.to, t, min(f, e.cap));
            if (d > 0) {
                e.cap -= d;
                adj[e.to][e.rev].cap += d;
                return d;
            }
        }
    }
    return 0;
}

int dinic(int s, int t) {
    int flow = 0;
    while (bfs(s, t)) {
        memset(iter, 0, sizeof(iter));
        int d;
        while ((d = dfs(s, t, INT_MAX)) > 0)
            flow += d;
    }
    return flow;
}
```

### 7.2 最小割

**最大流最小割定理**：最大流 = 最小割。

求最小割的具体方案：跑完最大流后做 BFS，从源点出发只走残余容量 > 0 的边，能到达的点集为 S，其余为 T。所有从 S 到 T 的满流边即为最小割边集。

**经典例题**：最大流（洛谷 P3376）
- 给定网络，求从源到汇的最大流量。直接套 Dinic 模板。

**经典应用**：
- 二分图最大匹配（源连左部，右部连汇，跑最大流）
- 最小割模型：项目选择、关键边问题

## 8. 最近公共祖先（LCA）- 倍增法

**核心思想**：预处理每个节点的 2^k 级祖先 `fa[u][k]`。查询时先将两节点跳到同一深度，再同时向上倍增跳跃，直到祖先相同。

**时间复杂度**：预处理 O(V log V)，每次查询 O(log V)

```cpp
const int LOG = 20;
int depth[MAXN], fa[MAXN][LOG];

void bfs(int root) {
    queue<int> q;
    memset(depth, 0, sizeof(depth));
    memset(fa, 0, sizeof(fa));
    depth[root] = 1;
    q.push(root);
    while (!q.empty()) {
        int u = q.front(); q.pop();
        for (int v : adj[u]) {
            if (depth[v]) continue;
            depth[v] = depth[u] + 1;
            fa[v][0] = u;
            for (int k = 1; k < LOG; k++)
                fa[v][k] = fa[fa[v][k-1]][k-1];
            q.push(v);
        }
    }
}

int lca(int u, int v) {
    if (depth[u] < depth[v]) swap(u, v);
    int diff = depth[u] - depth[v];
    for (int k = 0; k < LOG; k++)
        if ((diff >> k) & 1) u = fa[u][k];
    if (u == v) return u;
    for (int k = LOG - 1; k >= 0; k--) {
        if (fa[u][k] != fa[v][k]) {
            u = fa[u][k];
            v = fa[v][k];
        }
    }
    return fa[u][0];
}
```

**经典例题**：LCA（洛谷 P3379）
- 给定一棵有根树，多次查询两节点的最近公共祖先。

**扩展应用**：
- 树上两点距离：`dist(u,v) = depth[u] + depth[v] - 2 * depth[lca(u,v)]`
- 树上差分与路径统计

## 9. 欧拉路径与欧拉回路

### 概念

- **欧拉路径**（Euler Path）：经过每条边恰好一次的路径。
- **欧拉回路**（Euler Circuit）：起点和终点相同的欧拉路径。

### 存在条件

| 图类型 | 欧拉回路 | 欧拉路径 |
|--------|---------|---------|
| **无向图** | 所有顶点度数为偶数 | 恰好 0 或 2 个奇度顶点 |
| **有向图** | 所有顶点入度 = 出度 | 恰一个出度 - 入度 = 1（起点），恰一个入度 - 出度 = 1（终点），其余相等 |

前提：图是连通的（忽略孤立点）。

### Hierholzer 算法

**核心思想**：从起点出发 DFS，每走一条边就删除它。走到死胡同时回溯，回溯时将节点加入路径。最终路径反转即为欧拉路径/回路。

**时间复杂度**：O(V + E)

```cpp
// 有向图版本，使用邻接表 + 当前弧优化
vector<int> adj[MAXN];
int cur[MAXN]; // 当前弧
vector<int> path;

void hierholzer(int u) {
    while (cur[u] < (int)adj[u].size()) {
        int v = adj[u][cur[u]++];
        hierholzer(v);
    }
    path.push_back(u);
}

// 调用后 reverse(path.begin(), path.end()) 得到欧拉路径
```

**无向图版本**：

```cpp
vector<pair<int,int>> adj[MAXN]; // {to, edge_id}
bool used[MAXM]; // 边是否已使用
vector<int> path;

void hierholzer(int u) {
    while (!adj[u].empty()) {
        auto [v, id] = adj[u].back();
        adj[u].pop_back();
        if (used[id]) continue;
        used[id] = true;
        hierholzer(v);
    }
    path.push_back(u);
}
// 最后 reverse(path)
```

- 给定若干机票（有向图），找出字典序最小的行程。用 Hierholzer + multiset（按字典序贪心选边）。

### 例题 1：LC 743 - 网络延迟时间

**题意**：给定 n 个节点的有向加权图，从节点 k 发出信号，求信号到达所有节点的最短时间。若无法到达所有节点返回 -1。

**思路**：从 k 出发跑 Dijkstra 堆优化求单源最短路，答案为所有 dist 的最大值。

```cpp
class Solution {
public:
    int networkDelayTime(vector<vector<int>>& times, int n, int k) {
        const int INF = 0x3f3f3f3f;
        vector<vector<pair<int,int>>> adj(n + 1);
        for (auto& e : times)
            adj[e[0]].push_back({e[1], e[2]});

        vector<int> dist(n + 1, INF);
        vector<bool> vis(n + 1, false);
        priority_queue<pair<int,int>, vector<pair<int,int>>, greater<>> pq;
        dist[k] = 0;
        pq.push({0, k});

        while (!pq.empty()) {
            auto [d, u] = pq.top(); pq.pop();
            if (vis[u]) continue;
            vis[u] = true;
            for (auto [v, w] : adj[u]) {
                if (dist[u] + w < dist[v]) {
                    dist[v] = dist[u] + w;
                    pq.push({dist[v], v});
                }
            }
        }

        int ans = 0;
        for (int i = 1; i <= n; i++) {
            if (dist[i] == INF) return -1;
            ans = max(ans, dist[i]);
        }
        return ans;
    }
};
```

### 例题 2：LC 684 - 冗余连接

**题意**：一棵 n 个节点的树多了一条边变成图，找出删除哪条边能恢复为树。若有多条答案，返回输入中最后出现的那条。

**思路**：并查集逐条加边，第一次发现两端已在同一集合（形成环）的边即为答案。

```cpp
class Solution {
public:
    int fa[1001];

    int find(int x) {
        return fa[x] == x ? x : fa[x] = find(fa[x]);
    }

    vector<int> findRedundantConnection(vector<vector<int>>& edges) {
        for (int i = 1; i <= 1000; i++) fa[i] = i;
        for (auto& e : edges) {
            int fu = find(e[0]), fv = find(e[1]);
            if (fu == fv) return e;
            fa[fu] = fv;
        }
        return {};
    }
};
```

### 例题 3：LC 210 - 课程表 II

**题意**：给定 numCourses 门课程和先修关系 prerequisites，返回一种合法的学习顺序。若不可能完成所有课程则返回空数组。

**思路**：BFS 拓扑排序（Kahn 算法）。统计入度，入度为 0 的入队，逐步取出并削减后继入度。最终排序结果长度等于课程数则合法。

```cpp
class Solution {
public:
    vector<int> findOrder(int numCourses, vector<vector<int>>& prerequisites) {
        vector<vector<int>> adj(numCourses);
        vector<int> indeg(numCourses, 0);
        for (auto& p : prerequisites) {
            adj[p[1]].push_back(p[0]);
            indeg[p[0]]++;
        }

        queue<int> q;
        for (int i = 0; i < numCourses; i++)
            if (indeg[i] == 0) q.push(i);

        vector<int> order;
        while (!q.empty()) {
            int u = q.front(); q.pop();
            order.push_back(u);
            for (int v : adj[u]) {
                if (--indeg[v] == 0) q.push(v);
            }
        }

        if ((int)order.size() != numCourses) return {};
        return order;
    }
};
